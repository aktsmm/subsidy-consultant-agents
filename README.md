# 補助金コンサル AIエージェント

中小企業向け補助金コンサルティング業務を支援する **GitHub Copilot エージェントシステム** です。  
質問するだけで、対象診断・補助金選定・申請サポートまで AI が案内します。

---

## できること

| やりたいこと           | 聞き方の例                             | AI がやること                  |
| ---------------------- | -------------------------------------- | ------------------------------ |
| 対象か調べたい         | 「製造業・従業員30人で補助金使える？」 | 業種・規模から対象可否を即診断 |
| どの補助金がいいか     | 「会計ソフト導入に使える補助金は？」   | 最適な補助金を比較・提案       |
| 申請方法を知りたい     | 「IT導入補助金の申請フローを教えて」   | 必要書類・手順をステップで案内 |
| スケジュール確認       | 「次の締切はいつ？」                   | 締切・準備期間を逆算して提示   |
| 補助額シミュレーション | 「予算100万円で補助額いくら？」        | 補助率から自己負担額を計算     |

---

## 使い方

### 1. VS Code でこのプロジェクトを開く

### 2. Copilot Chat を起動（`Ctrl+Shift+I`）

### 3. 質問する

```
@subsidy-consultant 補助金について相談したい
```

AI の質問に答えるだけで、業種・規模・やりたいことに最適な補助金を提案します。

## 対応している補助金

| 補助金                   | 補助率   | おすすめの人                           |
| ------------------------ | -------- | -------------------------------------- |
| IT導入補助金             | 1/2〜4/5 | 会計ソフト・業務システムを導入したい   |
| ものづくり補助金         | 1/2〜2/3 | 製造設備・生産ラインに投資したい       |
| 小規模事業者持続化補助金 | 2/3      | HP作成・チラシ・展示会で販路開拓したい |
| 省力化投資補助金         | 1/2〜2/3 | ロボット・自動化で人手不足を解消したい |
| 事業承継・M&A補助金      | 1/2〜2/3 | 事業承継や M&A に伴う経費を補助        |

## エージェント構成

```
                    ┌─────────────────────┐
                    │ subsidy-consultant  │
                    │   (Orchestrator)    │
                    └──────────┬──────────┘
                               │ runSubagent
       ┌───────────┬───────────┼───────────┬───────────┬───────────┐
       ▼           ▼           ▼           ▼           ▼           ▼
┌─────────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐
│ eligibility │ │ subsidy │ │  app    │ │   web   │ │schedule │ │ answer  │
│   checker   │ │ selector│ │  guide  │ │researcher│ │ tracker │ │recorder │
└─────────────┘ └─────────┘ └─────────┘ └─────────┘ └─────────┘ └────┬────┘
                                                                     │ handoff
                                                                     ▼
                                                              ┌─────────────┐
                                                              │   answer    │
                                                              │  reviewer   │
                                                              └─────────────┘
```

| エージェント        | 役割                                           | 呼び出し方             |
| ------------------- | ---------------------------------------------- | ---------------------- |
| subsidy-consultant  | 司令塔。顧客対応を統括し専門エージェントに委譲 | `@subsidy-consultant`  |
| eligibility-checker | 補助金の対象者要件を診断                       | `@eligibility-checker` |
| subsidy-selector    | 最適な補助金を選定・比較                       | `@subsidy-selector`    |
| application-guide   | 申請フローと必要書類を案内                     | `@application-guide`   |
| web-researcher      | 補助金の最新情報を Web 検索で調査              | `@web-researcher`      |
| schedule-tracker    | 締切・スケジュールを管理                       | `@schedule-tracker`    |
| answer-recorder     | 回答を Markdown 形式で自動記録                 | `@answer-recorder`     |
| answer-reviewer     | 記録された回答の品質チェック・自動承認         | `@answer-reviewer`     |

## 注意事項

- ナレッジベースは **2026年1月時点** の情報です。申請前に必ず公式サイトで最新の公募要領を確認してください。

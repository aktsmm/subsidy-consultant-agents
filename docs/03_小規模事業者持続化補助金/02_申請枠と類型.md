# 小規模事業者持続化補助金 申請枠と類型

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [公式サイト](https://r6.jizokukahojokin.info/)

---

## 申請類型一覧

| 類型                   | 補助率 | 補助上限額 | 主な対象                     |
| ---------------------- | ------ | ---------- | ---------------------------- |
| 一般型（通常枠）       | 2/3    | 50万円     | 販路開拓全般                 |
| 一般型（賃金引上げ枠） | 2/3    | 200万円    | 最低賃金引上げ事業者         |
| 一般型（卒業枠）       | 2/3    | 200万円    | 従業員を増やす事業者         |
| 一般型（後継者支援枠） | 2/3    | 200万円    | アトツギ甲子園ファイナリスト |
| 一般型（創業枠）       | 2/3    | 200万円    | 過去3年以内の創業者          |
| 災害支援枠             | 2/3    | 200万円    | 被災事業者                   |
| 創業型                 | 2/3    | 200万円    | 創業後間もない事業者         |
| 共同・協業型           | 2/3    | 1,000万円  | 複数事業者の連携             |

---

## 1. 一般型（通常枠）

販路開拓や生産性向上の取り組みを支援します。

| 項目       | 内容                                                       |
| ---------- | ---------------------------------------------------------- |
| 補助率     | 2/3                                                        |
| 補助上限額 | 50万円                                                     |
| 対象経費   | 機械装置等費、広報費、ウェブサイト関連費、展示会等出展費等 |

### 特別枠（上限額の引き上げ）

| 特別枠       | 補助上限額 | 対象                               |
| ------------ | ---------- | ---------------------------------- |
| 賃金引上げ枠 | 200万円    | 事業場内最低賃金を引き上げる事業者 |
| 卒業枠       | 200万円    | 常時使用する従業員を増やす事業者   |
| 後継者支援枠 | 200万円    | アトツギ甲子園ファイナリスト       |
| 創業枠       | 200万円    | 過去3年以内に創業した事業者        |

### 活用例

| ケース     | 内容                               | 経費区分       |
| ---------- | ---------------------------------- | -------------- |
| チラシ制作 | 新商品のチラシ10,000部作成・配布   | 広報費         |
| 展示会出展 | 地域の展示会にブース出展           | 展示会等出展費 |
| 店舗改装   | 集客力向上のための店舗リニューアル | 委託・外注費   |

---

## 2. 災害支援枠

自然災害等からの事業再建を支援します。

| 項目       | 内容                              |
| ---------- | --------------------------------- |
| 補助率     | 2/3                               |
| 補助上限額 | 200万円                           |
| 対象       | 令和6年能登半島地震等の被災事業者 |

---

## 3. 創業型

創業初期の販路開拓や業務改善を支援します。

| 項目       | 内容                 |
| ---------- | -------------------- |
| 補助率     | 2/3                  |
| 補助上限額 | 200万円              |
| 対象       | 創業後間もない事業者 |

---

## 4. 共同・協業型

複数事業者の展示会、商談会等を活用する販路開拓を支援します。

| 項目       | 内容                                     |
| ---------- | ---------------------------------------- |
| 補助率     | 2/3                                      |
| 補助上限額 | 1,000万円（参加事業者数による）          |
| 対象       | 複数の小規模事業者が連携して行う販路開拓 |

---

## 対象経費の詳細

| 経費区分           | 例                       | 注意                    |
| ------------------ | ------------------------ | ----------------------- |
| 機械装置等費       | 製造機械、検査機器       |                         |
| 広報費             | チラシ、パンフレット作成 |                         |
| ウェブサイト関連費 | HP作成・改修             | **補助金総額の1/4まで** |
| 展示会等出展費     | 展示会・商談会への出展   |                         |
| 旅費               | 販路開拓のための出張費   |                         |
| 開発費             | 新商品の試作開発         |                         |
| 資料購入費         | 図書、資料               |                         |
| 借料               | 機器・会場のレンタル     |                         |
| 設備処分費         | 旧設備の処分             | **補助金総額の1/2まで** |
| 委託・外注費       | デザイン、店舗改装       |                         |

### 💡 ウェブサイト関連費の注意

> ウェブサイト関連費のみの申請は**原則不可**です。他の経費と組み合わせて申請してください。また、補助金総額の**1/4が上限**です（通常枠50万円の場合、最大12.5万円まで）。

---

## 申請枠の選び方フローチャート

```
[事業者の状況は？]
    │
    ├── 創業後3年以内 → 創業枠（200万円）
    │
    ├── 被災事業者 → 災害支援枠（200万円）
    │
    ├── 複数事業者で連携 → 共同・協業型（最大1,000万円）
    │
    └── 上記以外 → 一般型
         │
         ├── 最低賃金を引き上げる → 賃金引上げ枠（200万円）
         │
         ├── 従業員を増やす → 卒業枠（200万円）
         │
         ├── アトツギ甲子園ファイナリスト → 後継者支援枠（200万円）
         │
         └── 上記以外 → 通常枠（50万円）
```

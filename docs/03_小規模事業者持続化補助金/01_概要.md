## 制度の仕組み

### 補助金の目的

小規模事業者が「自ら経営計画を策定」し、地域の商工会議所・商工会の支援を受けながら行う、**販路開拓**などの取組を支援する補助金です。持続的な経営に向けた経営計画に基づく地道な販路開拓等の取り組みや、業務効率化の取り組みを支援します。

### 制度の特徴

| 特徴                               | 内容                                     |
| ---------------------------------- | ---------------------------------------- |
| **商工会議所・商工会の支援が必須** | 経営計画の策定段階から支援を受ける       |
| **販路開拓が中心**                 | チラシ、HP、展示会等の販路開拓費用が対象 |
| **小規模事業者限定**               | 従業員数の上限あり（5人〜20人）          |
| **比較的少額**                     | 補助上限額50万〜250万円（通常枠）        |

## 他補助金との比較

| 項目         | 小規模事業者持続化補助金     | IT導入補助金                 | ものづくり補助金       |
| ------------ | ---------------------------- | ---------------------------- | ---------------------- |
| 対象者       | 小規模事業者のみ             | 中小企業・小規模事業者       | 中小企業・小規模事業者 |
| 対象経費     | 販路開拓全般                 | ITツール（ソフトウェア）     | 設備投資全般           |
| 補助上限額   | 最大250万円                  | 最大450万円（通常枠）        | 最大4,000万円          |
| 申請プロセス | 商工会議所・商工会の支援必須 | IT導入支援事業者と連携       | 認定支援機関の確認必要 |
| ウェブサイト | 対象だが上限あり（1/4まで）  | 対象外（業務システムは対象） | 対象外                 |
| 審査難易度   | 比較的容易                   | 容易                         | 厳格                   |

## コンサルとしての主要チェックポイント

### 初回ヒアリング時

- [ ] 従業員数の確認（小規模事業者に該当するか）
- [ ] 管轄の商工会議所・商工会の確認
- [ ] 販路開拓したい内容の明確化
- [ ] 過去の持続化補助金利用履歴

### 申請準備時

- [ ] 経営計画書・補助事業計画書の策定
- [ ] 商工会議所・商工会からの「事業支援計画書」取得
- [ ] 特別枠の該当有無確認

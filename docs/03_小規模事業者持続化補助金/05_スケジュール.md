## 年間スケジュールイメージ

```
【通年】商工会議所・商工会への相談
  └── 経営相談は随時受付

【公募開始時】（年2〜4回）
  ├── 公募要領の確認
  ├── 商工会議所への相談予約
  └── 経営計画書のドラフト作成開始

【締切1ヶ月前】
  ├── 経営計画書・補助事業計画書の完成
  └── 事業支援計画書の発行依頼

【締切1〜2週間前】
  ├── 必要書類の最終確認
  └── 申請書類の提出（郵送 or 電子申請）

【採択後（約2ヶ月後）】
  ├── 採択通知
  ├── 交付申請
  └── 交付決定

【補助事業実施期間】
  ├── 販路開拓の実施
  ├── 経費の支出・証拠書類保管
  └── 実績報告

【事業終了後】
  └── 効果報告
```

## 申請準備の逆算スケジュール

### 締切2ヶ月前〜

| 残り日数   | やるべきこと                                   |
| ---------- | ---------------------------------------------- |
| **60日前** | 管轄の商工会議所・商工会に初回相談             |
| **50日前** | 経営計画書のドラフト作成開始                   |
| **40日前** | 補助事業計画書の作成                           |
| **30日前** | 事業支援計画書の発行依頼                       |
| **21日前** | 必要書類の収集（確定申告書、決算書等）         |
| **14日前** | 申請書類の最終チェック                         |
| **7日前**  | 郵送の場合は投函、電子申請の場合はアップロード |
| **当日**   | 申請締切（必着）                               |

> **⚠️ 注意**: 郵送の場合は「消印有効」ではなく「必着」です。余裕を持って発送してください。

## 審査〜補助金交付までの期間

| フェーズ             | 期間目安                          |
| -------------------- | --------------------------------- |
| 申請締切〜採択発表   | 約2ヶ月                           |
| 採択〜交付決定       | 約2〜4週間                        |
| 交付決定〜事業実施   | 公募要領記載の期間（約6ヶ月程度） |
| 事業終了〜実績報告   | 30日以内                          |
| 実績報告〜補助金交付 | 約1〜2ヶ月                        |

# 小規模事業者持続化補助金 対象者要件

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [公式サイト](https://r6.jizokukahojokin.info/)

---

## 対象となる小規模事業者

### 業種別の従業員数上限

| 業種                                   | 従業員数     |
| -------------------------------------- | ------------ |
| 商業・サービス業（宿泊業・娯楽業除く） | **5人以下**  |
| 宿泊業・娯楽業                         | **20人以下** |
| 製造業その他                           | **20人以下** |

> **注意**:「常時使用する従業員」は正社員だけでなく、パート・アルバイトで所定労働時間が正社員の3/4以上の者も含みます。ただし、日雇い・2ヶ月以内の雇用契約の者等は除きます。

## 対象となる法人形態

### 対象

| 法人形態     | 対象可否 |
| ------------ | -------- |
| 株式会社     | ✅       |
| 合名会社     | ✅       |
| 合資会社     | ✅       |
| 合同会社     | ✅       |
| 特例有限会社 | ✅       |
| 個人事業主   | ✅       |
| 医療法人     | ✅       |
| 社会福祉法人 | ✅       |
| NPO法人      | ✅       |

### 対象外

| 法人形態               | 対象可否                 |
| ---------------------- | ------------------------ |
| 一般社団法人・財団法人 | ❌                       |
| 医師・歯科医師         | ❌（開設者が個人の場合） |
| 学校法人               | ❌                       |
| 宗教法人               | ❌                       |

## 申請に必要な条件

### 必須条件

- [ ] 小規模事業者であること（上記の従業員数要件を満たす）
- [ ] 商工会議所または商工会の管轄地域内で事業を行っていること
- [ ] 持続的な経営に向けた経営計画を策定していること
- [ ] 「事業支援計画書」を商工会議所・商工会から発行してもらうこと

### 申請に必要な準備

| 項目                   | 内容                                       | 取得期間       |
| ---------------------- | ------------------------------------------ | -------------- |
| 確定申告書の写し       | 直近1期分                                  | 手元にあるはず |
| 貸借対照表・損益計算書 | 法人の場合、直近1期分                      | 手元にあるはず |
| 事業支援計画書         | 商工会議所・商工会が発行                   | 約1〜2週間     |
| 経営計画書             | 自社で策定（商工会議所の支援を受けながら） | 約2〜4週間     |

---

## 対象外となるケース

### 申請不可の条件

- [ ] 同一内容で持続化補助金と他の国の補助金を併用する場合
- [ ] 同一の公募回に複数箇所の商工会議所・商工会から重複して申請する場合
- [ ] 過去に持続化補助金で補助事業を実施し、「事業効果及び賃金引上げ等状況報告書」を提出していない場合

### 開業届未提出の場合

個人事業主で開業届を提出していない場合は申請できません。税務署への「個人事業の開業届出書」の提出が必要です。

## 業種別の適合チェック

### 飲食業（サービス業）

| チェック項目 | 確認内容                                           |
| ------------ | -------------------------------------------------- |
| 従業員数     | 5人以下か？（宿泊業・娯楽業は20人以下）            |
| 販路開拓     | メニュー開発、チラシ作成、HP作成等を予定しているか |
| 管轄地域     | 所在地の商工会議所・商工会はどこか                 |

### 小売業（商業）

| チェック項目 | 確認内容                                                 |
| ------------ | -------------------------------------------------------- |
| 従業員数     | 5人以下か？                                              |
| 販路開拓     | 新商品の開発、ECサイト構築、展示会出展等を予定しているか |
| 管轄地域     | 所在地がどの商工会議所・商工会の管轄か                   |

### 製造業

| チェック項目 | 確認内容                                                     |
| ------------ | ------------------------------------------------------------ |
| 従業員数     | 20人以下か？                                                 |
| 販路開拓     | 新製品の試作、展示会出展、パンフレット作成等を予定しているか |
| 設備投資     | 製造機械の導入も対象（ものづくり補助金との使い分けも検討）   |

## よくある質問

### Q: 商工会議所と商工会の違いは？

**A**: 事業所の所在地によって管轄が異なります。

| 地域           | 管轄団体   |
| -------------- | ---------- |
| 市部（主に）   | 商工会議所 |
| 町村部（主に） | 商工会     |

> 自分の管轄がわからない場合は、市区町村の窓口に確認してください。

### Q: 従業員数が増えて要件を超えたら？

**A**: 申請時点で小規模事業者の要件を満たしていれば申請可能です。事業実施中に従業員が増えても、補助金の返還は求められません。

### Q: 開業したばかりでも申請できますか？

**A**: はい、開業届を提出していれば申請可能です。「創業枠」を利用すると補助上限額が200万円に増額されます（過去3年以内の創業が条件）。

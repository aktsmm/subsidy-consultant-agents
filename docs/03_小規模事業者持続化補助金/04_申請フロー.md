# 小規模事業者持続化補助金 申請フロー

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [公式サイト](https://r6.jizokukahojokin.info/)

---

## 申請の全体フロー

```
STEP1: 事前準備・商工会議所との相談（2〜3ヶ月前）
  ├── 管轄の商工会議所・商工会に相談
  ├── 経営状況の棚卸し
  └── 補助事業のアイデア整理
    ↓
STEP2: 経営計画書・補助事業計画書の策定（1〜2ヶ月前）
  ├── 経営計画書の作成（商工会議所の支援を受けながら）
  ├── 補助事業計画書の作成
  └── 数値計画の策定
    ↓
STEP3: 事業支援計画書の発行依頼（3〜4週間前）
  └── 商工会議所・商工会に依頼
    ↓
STEP4: 申請書類の提出
  ├── 郵送または電子申請（jGrants）
  └── 締切日必着
    ↓
STEP5: 審査・採択決定（約2ヶ月）
  ├── 書面審査
  └── 採択通知
    ↓
STEP6: 交付申請・交付決定
  ├── 交付申請書の提出
  └── 交付決定通知の受領
    ↓
STEP7: 補助事業の実施
  ├── 販路開拓の実施 ※必ず交付決定後に行うこと
  ├── 経費の支出
  └── 証拠書類の保管
    ↓
STEP8: 実績報告
  ├── 事業実施の報告
  └── 経費の証拠書類提出
    ↓
STEP9: 補助金の請求・交付
```

---

## 各STEPの詳細

### STEP1: 商工会議所・商工会への相談

#### これが持続化補助金の最大の特徴です

他の補助金と異なり、**商工会議所・商工会の支援を受けることが必須**です。

```
【相談の流れ】
1. 管轄の商工会議所・商工会を確認
2. 窓口に連絡（電話またはメール）
3. 経営指導員との面談予約
4. 現在の経営状況と課題を説明
5. 補助事業のアイデアについてアドバイスを受ける
```

> **ポイント**: 初回相談は無料です。経営指導員は補助金申請のアドバイスだけでなく、経営全般の相談にも乗ってくれます。

---

### STEP2: 経営計画書・補助事業計画書の策定

#### 経営計画書の記載項目

| 項目                                     | 記載内容                   |
| ---------------------------------------- | -------------------------- |
| 企業概要                                 | 事業内容、沿革、従業員数等 |
| 顧客ニーズと市場の動向                   | ターゲット顧客、市場環境   |
| 自社や自社の提供する商品・サービスの強み | 競合との差別化ポイント     |
| 経営方針・目標と今後のプラン             | 中長期の経営ビジョン       |

#### 補助事業計画書の記載項目

| 項目                 | 記載内容                   |
| -------------------- | -------------------------- |
| 補助事業で行う事業名 | 簡潔な事業名（30字以内）   |
| 販路開拓等の取組内容 | 具体的な実施内容           |
| 業務効率化の取組内容 | 業務効率化も行う場合       |
| 補助事業の効果       | 売上・利益等への効果見込み |

---

### STEP3: 事業支援計画書の発行

| 項目     | 内容                             |
| -------- | -------------------------------- |
| 発行元   | 管轄の商工会議所・商工会         |
| 費用     | 無料                             |
| 所要期間 | 約1〜2週間                       |
| ⚠️ 注意  | 締切間際は混雑するため早めに依頼 |

---

### STEP4: 申請書類の提出

#### 必要書類一覧

| 書類                                  | 法人 | 個人 | 備考             |
| ------------------------------------- | :--: | :--: | ---------------- |
| 経営計画書兼補助事業計画書（様式2-1） |  ✅  |  ✅  |                  |
| 補助事業計画書（様式3-1）             |  ✅  |  ✅  |                  |
| 事業支援計画書（様式4）               |  ✅  |  ✅  | 商工会議所が発行 |
| 貸借対照表・損益計算書                |  ✅  |  -   | 直近1期分        |
| 確定申告書                            |  -   |  ✅  | 直近1期分        |
| 開業届の写し                          |  -   |  ✅  | 創業枠の場合     |

#### 申請方法

| 方法                | 説明                           |
| ------------------- | ------------------------------ |
| 郵送                | 締切日必着（消印有効ではない） |
| 電子申請（jGrants） | gBizIDプライムが必要           |

> **推奨**: 電子申請（jGrants）の方が処理が早く、書類の不備も少なくなります。

## 💡 持続化補助金特有の注意点

### 1. ウェブサイト関連費の上限

ウェブサイト関連費（HP作成・改修）は補助金総額の**1/4が上限**です。

```
【計算例】
通常枠（補助上限50万円）の場合:
  ウェブサイト関連費の上限 = 50万円 × 1/4 = 12.5万円

賃金引上げ枠（補助上限200万円）の場合:
  ウェブサイト関連費の上限 = 200万円 × 1/4 = 50万円
```

### 2. 経営計画書が審査の肝

事業計画の質が採択に直結します。以下のポイントを押さえましょう：

- **市場分析**: 「誰に」「何を」売るのかを明確に
- **強みの活用**: 自社の強みをどう活かすかを具体的に
- **数値目標**: 売上増加額を具体的な数値で示す
- **実現可能性**: なぜこの取り組みが成功するのかの根拠

### 3. 補助事業の実施時期

交付決定後に事業を実施する必要があります。事前着手は原則認められません。

## よくある失敗パターン

| 失敗                               | 対策                         |
| ---------------------------------- | ---------------------------- |
| 事業支援計画書の発行が間に合わない | 締切の3〜4週間前に依頼       |
| 経営計画書が抽象的                 | 具体的な数値と根拠を記載     |
| ウェブサイト費だけの申請           | 他の経費と組み合わせる       |
| 郵送が締切に届かない               | 余裕を持って発送 or 電子申請 |
| 交付決定前に事業を開始             | 交付決定通知を必ず確認       |

# ものづくり補助金 概要

> **情報更新日**: 2026年1月28日  
> **正式名称**: ものづくり・商業・サービス生産性向上促進補助金  
> **公式サイト**: [ものづくり補助金総合サイト](https://portal.monodukuri-hojo.jp/)

---

## 制度の仕組み

### 補助金の目的

中小企業・小規模事業者等が今後複数年にわたり相次いで直面する制度変更（働き方改革、被用者保険の適用拡大、賃上げ、インボイス導入等）等に対応するため、中小企業等が取り組む**革新的な製品・サービスの開発**、**生産プロセス・サービス提供方法の改善**に必要な設備投資等を支援する補助金です。

### 制度の関係者

```
┌─────────────────────────────────────────────────────────────┐
│                    事務局（全国中小企業団体中央会）            │
│  ・公募要領の公開                                           │
│  ・交付申請の審査・交付決定                                  │
│  ・実績報告の確認・補助金交付                                │
└─────────────────────────────────────────────────────────────┘
          ▲                              ▲
          │ 申請                          │ 交付申請
          │                              │
┌─────────────────────┐          ┌─────────────────────┐
│ 認定経営革新等         │◀────────▶│ 中小企業・小規模事業者 │
│ 支援機関              │  支援     │ （補助事業者）        │
│                      │          │                      │
│ ・事業計画の策定支援   │          │ ・事業計画の策定      │
│ ・金融機関等          │          │ ・設備投資の実施      │
│ ・商工会議所          │          │ ・効果報告            │
└─────────────────────┘          └─────────────────────┘
```

### 補助対象となる経費

| 対象経費                 | 例                                        |
| ------------------------ | ----------------------------------------- |
| 機械装置・システム構築費 | 製造機械、金型、検査装置、CAD/CAMシステム |
| 技術導入費               | 特許権、ライセンス費用                    |
| 専門家経費               | コンサルティング費用                      |
| 運搬費                   | 設備の輸送費                              |
| クラウドサービス利用費   | SaaS利用料                                |
| 原材料費                 | 試作品の材料費                            |
| 外注費                   | 加工・設計の外注                          |
| 知的財産権等関連経費     | 特許出願費用                              |

## 他補助金との比較

| 項目       | ものづくり補助金     | IT導入補助金                 | 省力化投資補助金 |
| ---------- | -------------------- | ---------------------------- | ---------------- |
| 対象経費   | 設備投資全般         | ITツール（ソフトウェア）中心 | ハードウェア中心 |
| 補助上限額 | 最大4,000万円        | 最大450万円（通常枠）        | 最大1億円        |
| 審査       | 事業計画の審査が厳格 | 比較的シンプル               | カタログ型は容易 |
| 採択率     | 約40〜50%            | 約50〜60%                    | カタログ型は高い |
| 主な対象   | 製造業・サービス業   | 全業種                       | 人手不足業種     |

## コンサルとしての主要チェックポイント

### 初回ヒアリング時

- [ ] 業種・従業員数・資本金の確認（中小企業要件）
- [ ] 投資したい設備・システムの明確化
- [ ] gBizIDプライムの取得状況
- [ ] 認定経営革新等支援機関との連携状況
- [ ] 過去のものづくり補助金利用履歴
- [ ] 賃上げ計画の確認（基本要件）

### 申請準備時

- [ ] 事業計画書の策定（革新性・実現可能性・事業効果）
- [ ] 認定支援機関の確認書取得
- [ ] 必要書類の準備（決算書、確定申告書等）
- [ ] jGrantsでの電子申請準備

### 交付決定後

- [ ] 契約・発注のタイミング管理（交付決定後に契約すること）
- [ ] 実績報告期限の把握
- [ ] 効果報告（年次）のスケジュール管理

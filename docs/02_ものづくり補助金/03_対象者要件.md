# ものづくり補助金 対象者要件

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [ものづくり補助金公式サイト](https://portal.monodukuri-hojo.jp/)

---

## 対象となる事業者

### 中小企業の定義

| 業種                     | 資本金        | 従業員数  |
| ------------------------ | ------------- | --------- |
| 製造業、建設業、運輸業等 | 3億円以下     | 300人以下 |
| 卸売業                   | 1億円以下     | 100人以下 |
| サービス業               | 5,000万円以下 | 100人以下 |
| 小売業                   | 5,000万円以下 | 50人以下  |

> **注意**: 資本金**または**従業員数のいずれかを満たせば対象です。

### 小規模事業者の定義

| 業種                                   | 従業員数     |
| -------------------------------------- | ------------ |
| 商業・サービス業（宿泊業・娯楽業除く） | **5人以下**  |
| 宿泊業・娯楽業                         | **20人以下** |
| 製造業その他                           | **20人以下** |

> 小規模事業者は補助率が優遇されます（2/3）。

---

## 基本要件（すべて必須）

以下の3つの基本要件をすべて満たす**事業計画**を策定する必要があります。

### 1. 付加価値額の増加

```
付加価値額 = 営業利益 + 人件費 + 減価償却費

要件: 年率平均3%以上の増加
```

### 2. 給与支給総額の増加

```
要件: 年率平均1.5%以上の増加
```

### 3. 事業場内最低賃金

```
要件: 地域別最低賃金 + 30円以上
```

> **重要**: これらは「計画達成の見込み」で審査されますが、事業完了後の効果報告で達成状況が確認されます。

## 必須準備事項

### 1. gBizIDプライムの取得

| 項目     | 内容                                             |
| -------- | ------------------------------------------------ |
| 概要     | 国の行政手続きを電子申請で行うための共通認証ID   |
| 費用     | 無料                                             |
| 取得期間 | 申請から約2〜3週間                               |
| URL      | [gBizID](https://gbiz-id.go.jp/)                 |
| 注意事項 | 印鑑証明書の郵送が必要（オンラインのみでは不可） |

### 2. 認定経営革新等支援機関の確認書

| 項目     | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 概要     | 事業計画の実効性を支援機関が確認した書類                     |
| 対象機関 | 銀行、信用金庫、商工会議所、税理士・公認会計士事務所等       |
| 費用     | 支援機関により異なる（無料〜数万円程度）                     |
| 必要時期 | 申請締切の2週間前までに依頼するのが安全                      |
| 検索     | [認定経営革新等支援機関検索](https://www.ninteishien.go.jp/) |

## 対象外となる事業者

以下に該当する場合は申請できません：

- [ ] 大企業の子会社（みなし大企業）
- [ ] 過去3年間にものづくり補助金の交付決定を受け、事業完了していない者
- [ ] 申請時に税金の未納がある者
- [ ] 反社会的勢力に関与している者
- [ ] 過去にものづくり補助金で不正受給があった者

### みなし大企業の判定基準

以下のいずれかに該当する場合、「みなし大企業」として対象外：

1. 資本金または出資金の大企業による出資比率が50%以上
2. 大企業の役員が役員総数の過半数を占める
3. 大企業からの100%出資子会社

## 業種別の適合チェック

### 製造業

| チェック項目 | 確認内容                               |
| ------------ | -------------------------------------- |
| 資本金       | 3億円以下か？                          |
| 従業員数     | 300人以下か？                          |
| 設備投資     | 革新的な製品開発に必要か？             |
| 事業計画     | 付加価値額3%以上増加の見込みがあるか？ |

### サービス業

| チェック項目 | 確認内容                         |
| ------------ | -------------------------------- |
| 資本金       | 5,000万円以下か？                |
| 従業員数     | 100人以下か？                    |
| 設備投資     | サービス提供方法の改善に必要か？ |
| 事業計画     | 生産性向上の定量的根拠があるか？ |

### 小売業・卸売業

| チェック項目 | 確認内容                             |
| ------------ | ------------------------------------ |
| 資本金       | 小売5,000万円以下、卸売1億円以下か？ |
| 従業員数     | 小売50人以下、卸売100人以下か？      |
| 設備投資     | 新商品開発や販売方法の革新に必要か？ |

## 加点要件（審査で有利になる項目）

### 主要な加点項目

| 加点項目                 | 内容                           | 取得難易度 |
| ------------------------ | ------------------------------ | ---------- |
| 経営革新計画の承認       | 都道府県知事の承認を受けた計画 | やや高い   |
| パートナーシップ構築宣言 | ポータルサイトでの公開         | 低い       |
| 事業継続力強化計画の認定 | BCP策定計画の認定              | 中程度     |
| 賃上げ加点               | 給与支給総額の増加率が高い     | 計画次第   |
| 被災事業者加点           | 自然災害等の被災事業者         | 該当者のみ |

### おすすめの加点対策

```
【優先度高・取得容易】
1. パートナーシップ構築宣言（Webで登録するだけ）
2. 賃上げ加点（事業計画で反映）

【優先度中・やや時間が必要】
3. 事業継続力強化計画（策定・申請に1〜2ヶ月）
4. 経営革新計画（都道府県の承認に2〜3ヶ月）
```

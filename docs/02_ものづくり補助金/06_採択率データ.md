## 採択される事業計画の傾向

### 高い評価を受けるポイント

| 評価項目   | 高得点のポイント                             |
| ---------- | -------------------------------------------- |
| 革新性     | 従来技術との明確な差別化、業界初の取り組み   |
| 実現可能性 | 具体的な実施体制、技術的裏付け、スケジュール |
| 事業効果   | 定量的な数値目標、売上・利益の増加見込み     |
| 社会的意義 | 地域経済への貢献、SDGs・脱炭素への対応       |

### 不採択になりやすいパターン

| パターン           | 説明                                           |
| ------------------ | ---------------------------------------------- |
| 単なる設備の更新   | 革新性がない（老朽化した機械の単純な買い替え） |
| 事業計画が抽象的   | 「売上を増やす」だけで具体的数値がない         |
| 市場分析が不足     | 顧客ニーズや競合分析が甘い                     |
| 数値の根拠が不明   | 付加価値額増加の計算根拠がない                 |
| 自社の強みが不明確 | なぜ自社がこの事業を行うのかの説明が弱い       |

## 加点項目と採択率の関係

### 加点の効果

| 加点項目数 | 採択率目安 |
| ---------- | ---------- |
| 0項目      | 約30〜35%  |
| 1〜2項目   | 約40〜50%  |
| 3項目以上  | 約55〜65%  |

> **結論**: 加点項目は可能な限り取得することを推奨します。特に「パートナーシップ構築宣言」は登録するだけで取得可能です。

# ものづくり補助金 申請枠と類型

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [ものづくり補助金公式サイト](https://portal.monodukuri-hojo.jp/)

---

## 申請枠一覧

| 申請枠                       | 補助率             | 補助上限額       | 主な対象                   |
| ---------------------------- | ------------------ | ---------------- | -------------------------- |
| 製品・サービス高付加価値化枠 | 中小1/2、小規模2/3 | 750万〜4,000万円 | 革新的な製品・サービス開発 |
| グローバル枠                 | 中小1/2、小規模2/3 | 3,000万円        | 海外展開・インバウンド対応 |

---

## 1. 製品・サービス高付加価値化枠

新製品・新サービスの開発の設備投資を支援します。

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| 対象       | 革新的な製品・サービスの開発に必要な設備投資       |
| 補助率     | 中小企業1/2、小規模事業者2/3                       |
| 補助上限額 | 従業員規模により750万円〜4,000万円                 |
| 対象経費   | 機械装置・システム構築費、技術導入費、専門家経費等 |

### 補助上限額の詳細

| 従業員数 | 補助上限額 |
| -------- | ---------- |
| 5人以下  | 750万円    |
| 6〜20人  | 1,500万円  |
| 21〜50人 | 3,000万円  |
| 51人以上 | 4,000万円  |

### 想定ケース

| 業種   | 活用例                                       |
| ------ | -------------------------------------------- |
| 製造業 | 新型の精密加工機を導入し、高精度部品の量産化 |
| 食品業 | AI搭載の品質検査装置で不良率を大幅削減       |
| 建設業 | 3Dプリンターによる建築模型・部材の内製化     |

---

## 2. グローバル枠

海外事業に伴う設備・システム投資を支援します。

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| 対象       | 海外市場への挑戦、インバウンド対応の強化           |
| 補助率     | 中小企業1/2、小規模事業者2/3                       |
| 補助上限額 | 3,000万円                                          |
| 対象経費   | 機械装置・システム構築費、海外旅費、通訳・翻訳費等 |

### 要件

- 海外への直接投資に関する事業計画であること
- または海外市場開拓に関する事業計画であること
- インバウンド対応の場合は訪日外国人向けサービス改善であること

### 想定ケース

| パターン     | 活用例                           |
| ------------ | -------------------------------- |
| 海外直接投資 | 海外工場の設備導入費用           |
| 海外市場開拓 | 海外展示会用の試作品製造設備     |
| インバウンド | 多言語対応サービスシステムの構築 |

---

## 申請枠の選び方フローチャート

```
[投資内容は？]
    │
    ├── 国内向けの製品・サービス開発 → 製品・サービス高付加価値化枠
    │
    └── 海外展開・インバウンド対応 → グローバル枠
```

---

## 補助対象経費の詳細

### 共通で利用可能な経費

| 経費区分                 | 内容                           | 注意事項                     |
| ------------------------ | ------------------------------ | ---------------------------- |
| 機械装置・システム構築費 | 製造機械、検査装置、システム等 | 単価50万円以上（税抜）が目安 |
| 技術導入費               | 特許権、ライセンス料           |                              |
| 専門家経費               | コンサルティング費用           | 日額上限あり                 |
| 運搬費                   | 設備の輸送費                   |                              |
| クラウドサービス利用費   | SaaS等の利用料                 | 最大2年分                    |
| 原材料費                 | 試作品の材料費                 |                              |
| 外注費                   | 加工・設計の外注               | 補助対象経費総額の1/2以下    |
| 知的財産権等関連経費     | 特許出願費用                   |                              |

### グローバル枠のみ利用可能な経費

| 経費区分     | 内容                 |
| ------------ | -------------------- |
| 海外旅費     | 海外渡航費、宿泊費   |
| 通訳・翻訳費 | 通訳者費用、翻訳費用 |

---

## 補助対象外の経費

以下は補助対象外となります：

- 土地・建物の取得費
- 自動車の購入費（公道を走行するもの）
- 汎用性の高い機器（PC、プリンター等）※事業専用で使用する場合を除く
- 人件費・通信費・光熱費等の間接経費
- 消耗品費

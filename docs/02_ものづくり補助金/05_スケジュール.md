## 年間スケジュールイメージ

```
【1月〜3月】事前準備期間
  ├── gBizIDプライム取得
  ├── 認定経営革新等支援機関との事前相談
  ├── 加点項目の取得開始
  └── 事業計画のドラフト作成

【春〜秋】公募・申請期間（年3〜4回）
  ├── 公募要領公開
  ├── 事業計画書の完成
  ├── 認定支援機関の確認書取得
  └── jGrantsで電子申請

【採択後】実施期間（約10ヶ月）
  ├── 交付決定
  ├── 設備の契約・発注（※交付決定後！）
  ├── 設備の導入・事業実施
  └── 実績報告

【毎年】効果報告（5年間）
  ├── 付加価値額の報告
  ├── 給与支給総額の報告
  └── 事業場内最低賃金の報告
```

## 申請準備の逆算スケジュール

### 締切2ヶ月前〜

| 残り日数   | やるべきこと                           |
| ---------- | -------------------------------------- |
| **60日前** | gBizIDプライム取得申請（未取得の場合） |
| **60日前** | 認定支援機関への初回相談               |
| **45日前** | 事業計画書のドラフト作成開始           |
| **45日前** | 加点項目の取得手続き開始               |
| **30日前** | 事業計画書のレビュー（支援機関と）     |
| **21日前** | 認定支援機関の確認書の依頼             |
| **14日前** | 添付書類の準備（決算書、労働者名簿等） |
| **7日前**  | jGrantsでの申請書類アップロード        |
| **3日前**  | 最終確認・申請提出                     |
| **当日**   | 申請締切（通常17:00）                  |

> **⚠️ 注意**: 締切日当日はシステムが混雑するため、3日前までの提出を推奨します。

## 効果報告のスケジュール

### 報告期間

交付決定日の属する年度の翌年度から5年間、毎年報告が必要です。

```
例: 2026年6月に交付決定の場合

2027年度: 第1回効果報告
2028年度: 第2回効果報告
2029年度: 第3回効果報告
2030年度: 第4回効果報告
2031年度: 第5回効果報告
```

> **重要**: 効果報告期限を過ぎると補助金の返還を求められる可能性があります。

# ものづくり補助金 注意点とよくある失敗

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [ものづくり補助金公式サイト](https://portal.monodukuri-hojo.jp/)

---

## 💡 最重要注意事項

### 1. 交付決定前の発注禁止

> **⚠️ 交付決定通知を受ける前に、契約・発注・支払いを行った経費は一切補助対象外です。**

これは最も多い失敗パターンです。

```
【よくある失敗例】
・採択発表を見て安心し、交付決定通知前に設備を発注してしまった
・「内定」を「交付決定」と勘違いして発注してしまった
・納期の都合で先に発注してしまった

【正しい流れ】
採択発表 → 交付決定通知書の受領 → 契約・発注 → 納品 → 支払い
```

### 2. 効果報告の義務（5年間）

補助金を受け取った後、**毎年1回、5年間**にわたり効果報告が必要です。

- 付加価値額、給与支給総額、事業場内最低賃金の実績を報告
- 未報告の場合、補助金の返還を求められる可能性あり
- 基本要件を達成できなかった場合も返還のリスクあり

### 3. 単価50万円未満の設備は対象外（原則）

機械装置・システム構築費は**単価50万円以上（税抜）**が基本です。少額の備品購入には向いていません。

## よくある失敗パターン

### 申請準備段階

| 失敗                               | 詳細                                     | 対策                                 |
| ---------------------------------- | ---------------------------------------- | ------------------------------------ |
| gBizID取得が間に合わない           | 申請から取得まで2〜3週間必要             | 申請の1ヶ月以上前に取得開始          |
| 認定支援機関の確認書が間に合わない | 支援機関の繁忙期は対応が遅れる           | 2〜3週間前に依頼                     |
| 事業計画が抽象的すぎる             | 「革新的な製品を開発する」だけでは不十分 | 具体的な数値・技術・市場を記載       |
| 加点項目を取得していない           | 採択率が大幅に下がる                     | パートナーシップ構築宣言は最低限取得 |

### 事業実施段階

| 失敗                 | 詳細                              | 対策                               |
| -------------------- | --------------------------------- | ---------------------------------- |
| 見積書が1社のみ      | 50万円以上は2社以上の見積りが必要 | 早めに複数社から見積りを取得       |
| 証拠書類の不備       | 領収書や振込明細が揃っていない    | 支払い時に必ず証拠書類を保管       |
| 事業期間の超過       | 実績報告期限に間に合わない        | 設備納期を考慮したスケジュール管理 |
| 補助対象外経費の計上 | 汎用PCやソフトウェアを計上        | 対象経費を事前に確認               |

### 実績報告段階

| 失敗             | 詳細                       | 対策                         |
| ---------------- | -------------------------- | ---------------------------- |
| 経費の変更未申請 | 計画と異なる経費を使用     | 計画変更は事前に事務局に相談 |
| 写真の不備       | 設備導入前の写真がない     | 導入前に「before」写真を撮影 |
| 報告期限の超過   | 30日以内に報告できなかった | カレンダーにリマインドを設定 |

## 不正受給の事例と処分

### 過去に確認された不正事例

| 不正の種類   | 事例                                 | 処分             |
| ------------ | ------------------------------------ | ---------------- |
| 架空契約     | 実際に設備を導入していないのに報告   | 全額返還＋加算金 |
| 経費の水増し | 実際の支払額を超える金額で報告       | 全額返還＋加算金 |
| 目的外使用   | 補助対象事業と異なる目的で設備を使用 | 全額返還         |
| 二重申請     | 他の補助金と同一経費で申請           | 全額返還＋加算金 |

### 処分内容

- **補助金の全額返還**
- **加算金の支払い**（延滞金に相当、年率10.95%）
- **今後の申請資格の停止**
- **事業者名の公表**（令和3年度以降）

> **⚠️ 不正受給は刑事罰（補助金適正化法違反）の対象にもなり得ます。**

## FAQ

### Q1: 他の補助金と併用できますか？

**A**: 同一の補助対象経費（設備等）に対して複数の補助金を併用することはできません。ただし、異なる経費であれば他の補助金（IT導入補助金等）との併用は可能です。

### Q2: リースでの設備導入は対象ですか？

**A**: リースは原則対象外です。ファイナンスリースの場合、一定の条件を満たせば対象となる場合がありますが、公募要領で確認が必要です。

### Q3: 個人事業主でも申請できますか？

**A**: はい、中小企業・小規模事業者の要件を満たす個人事業主であれば申請可能です。

### Q4: 採択されなかった場合、次回申請できますか？

**A**: はい、不採択の場合は次回以降の公募に再度申請可能です。事業計画を見直して再チャレンジすることを推奨します。

### Q5: 事業計画書の分量はどのくらいが適切ですか？

**A**: A4で10ページ以内が目安です。審査員が短時間で読むため、要点を絞った簡潔な記載が重要です。過度に長い計画書はかえって評価が下がる場合があります。

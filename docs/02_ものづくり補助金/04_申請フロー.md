# ものづくり補助金 申請フロー

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [ものづくり補助金公式サイト](https://portal.monodukuri-hojo.jp/)

---

## 申請の全体フロー

```
STEP1: 事前準備（1〜3ヶ月前）
  ├── gBizIDプライム取得
  ├── 認定経営革新等支援機関への相談
  └── 加点項目の取得（パートナーシップ構築宣言等）
    ↓
STEP2: 事業計画の策定（1〜2ヶ月前）
  ├── 事業計画書の作成
  ├── 数値計画（付加価値額・給与等）の策定
  └── 認定支援機関の確認書取得
    ↓
STEP3: 交付申請（電子申請）
  ├── jGrantsでの申請書類アップロード
  ├── 事業計画書・各種添付書類の提出
  └── 認定支援機関の確認書の添付
    ↓
STEP4: 審査・交付決定（約2ヶ月）
  ├── 書類審査（外部有識者による審査）
  └── 交付決定通知
    ↓
STEP5: 補助事業の実施
  ├── 設備の契約・発注 ※必ず交付決定後に行うこと
  ├── 設備の納品・導入
  └── 支払い完了
    ↓
STEP6: 実績報告
  ├── 事業で使った経費の証拠書類を提出
  ├── 設備導入の証拠写真
  └── 成果報告
    ↓
STEP7: 確定検査・補助金交付
  ├── 事務局による確認
  └── 補助金の振込
    ↓
STEP8: 効果報告（年次・5年間）
  ├── 付加価値額の報告
  ├── 給与支給総額の報告
  └── 事業場内最低賃金の報告
```

---

## 各STEPの詳細

### STEP1: 事前準備

#### gBizIDプライムの取得

| 項目     | 内容                                    |
| -------- | --------------------------------------- |
| 目的     | jGrants（電子申請システム）へのログイン |
| 取得期間 | 約2〜3週間                              |
| 必要書類 | 印鑑証明書（発行から3ヶ月以内）         |
| 費用     | 無料                                    |
| ⚠️ 注意  | 法人の場合、代表者での取得が必要        |

#### 認定支援機関への相談

```
【認定支援機関の例】
├── 地域の金融機関（銀行、信用金庫）
├── 商工会議所・商工会
├── 税理士事務所・公認会計士事務所
└── 中小企業診断士
```

> **ポイント**: 金融機関が認定支援機関を兼ねている場合、融資相談も同時に進められます。

---

### STEP2: 事業計画の策定

#### 事業計画書の構成

| セクション       | 内容                               | ページ目安 |
| ---------------- | ---------------------------------- | ---------- |
| 会社概要         | 沿革、事業内容、組織体制           | 1〜2P      |
| 事業の背景・課題 | 現状の課題と解決の方向性           | 1〜2P      |
| 具体的な取組内容 | 導入設備、実施体制、スケジュール   | 3〜5P      |
| 革新性           | 他社にない独自性、新規性           | 1〜2P      |
| 事業効果         | 付加価値額、売上、生産性の数値計画 | 1〜2P      |

#### 数値計画の作成

```
【必須の数値目標】
├── 付加価値額: 年率平均3%以上増加
├── 給与支給総額: 年率平均1.5%以上増加
└── 事業場内最低賃金: 地域別最低賃金+30円以上

【計算式】
付加価値額 = 営業利益 + 人件費 + 減価償却費
```

---

### STEP3: 交付申請

#### 申請に必要な書類

| 書類                     | 取得先     | 注意事項               |
| ------------------------ | ---------- | ---------------------- |
| 事業計画書               | 自社作成   | A4で10枚以内推奨       |
| 認定支援機関の確認書     | 支援機関   | 2週間前に依頼          |
| 決算書（直近2期分）      | 自社保管   | 貸借対照表、損益計算書 |
| 確定申告書（個人事業主） | 保管分     | 控えのコピー           |
| 労働者名簿               | 自社作成   | 従業員数の確認用       |
| 法人事業概況説明書       | 税務署控え | 法人のみ               |

#### jGrantsでの申請手順

1. gBizIDでjGrantsにログイン
2. 「ものづくり補助金」の公募を検索
3. 申請フォームに必要事項を入力
4. 添付書類をアップロード
5. 申請内容を確認して提出

---

### STEP5: 補助事業の実施

#### 💡 最重要注意事項

> **⚠️ 交付決定前に契約・発注・支払いを行った経費は補助対象外です！**

```
【OK】交付決定日: 6月1日 → 契約日: 6月5日 → 納品: 8月 → 支払い: 9月
【NG】交付決定日: 6月1日 → 契約日: 5月15日（交付決定前は不可）
```

#### 経費管理のポイント

- すべての経費について「見積書→契約書→納品書→請求書→支払い証明」を揃える
- 50万円以上の案件は**2社以上の見積り**が必要
- クレジットカード払いは原則不可（銀行振込を推奨）
- 現金払いは証拠書類の管理に注意

---

### STEP6: 実績報告

#### 報告に必要な書類

| 書類       | 内容                               |
| ---------- | ---------------------------------- |
| 実績報告書 | 事業の実施状況                     |
| 経費明細   | 使った経費の一覧                   |
| 証拠書類   | 見積書、契約書、請求書、振込明細等 |
| 写真       | 設備導入の before/after            |
| 成果物     | 開発した製品・サービスの説明資料   |

---

### STEP8: 効果報告（5年間）

#### 報告内容

| 報告項目         | 内容           |
| ---------------- | -------------- |
| 付加価値額       | 計画との対比   |
| 給与支給総額     | 計画との対比   |
| 事業場内最低賃金 | 基準達成の確認 |

> **注意**: 効果報告は毎年1回、5年間にわたり実施が必要です。未報告の場合、補助金の返還を求められる可能性があります。

## よくある失敗パターン

| 失敗                           | 対策                               |
| ------------------------------ | ---------------------------------- |
| gBizID取得が間に合わない       | 申請の3週間以上前に取得開始        |
| 支援機関の確認書が間に合わない | 2〜3週間前に依頼                   |
| 交付決定前に発注してしまった   | 交付決定通知を必ず確認してから契約 |
| 見積書が1社しかない            | 50万円以上は必ず2社以上取得        |
| 効果報告を忘れる               | スケジュール管理で毎年リマインド   |

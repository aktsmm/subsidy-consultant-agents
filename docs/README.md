## 補助金比較表

| 補助金                   | 補助率   | 補助上限額    | 主な対象経費                 | 採択率   |
| ------------------------ | -------- | ------------- | ---------------------------- | -------- |
| IT導入補助金             | 1/2〜4/5 | 最大450万円   | ITツール（ソフトウェア）     | 50〜60%  |
| ものづくり補助金         | 1/2〜2/3 | 最大4,000万円 | 設備投資全般                 | 40〜50%  |
| 小規模事業者持続化補助金 | 2/3      | 最大250万円   | 販路開拓（チラシ、展示会等） | 50〜60%  |
| 省力化投資補助金         | 1/2〜2/3 | 最大1億円     | ロボット・自動化機器         | 随時受付 |
| 事業承継・M&A補助金      | 1/2〜2/3 | 最大800万円   | 事業承継に伴う経費           | 60〜70%  |

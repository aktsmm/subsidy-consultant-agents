# 補助金コンサル AIエージェント

> 💬 **「補助金使いたいけど、何から始めればいい？」**  
> → AIに聞くだけで、**対象診断・補助金選定・申請サポート**まで全部やってくれます！

---

## ✨ こんなことができます

| やりたいこと                  | 聞き方の例                             | AIがやってくれること           |
| ----------------------------- | -------------------------------------- | ------------------------------ |
| 🔍 **対象か調べたい**         | 「製造業・従業員30人で補助金使える？」 | 業種・規模から対象可否を即診断 |
| 💡 **どの補助金がいいか**     | 「会計ソフト導入に使える補助金は？」   | 最適な補助金を比較・提案       |
| 📋 **申請方法を知りたい**     | 「IT導入補助金の申請フローを教えて」   | 必要書類・手順をステップで案内 |
| 📅 **スケジュール確認**       | 「次の締切はいつ？」                   | 締切・準備期間を逆算して提示   |
| 💰 **補助額シミュレーション** | 「予算100万円で補助額いくら？」        | 補助率から自己負担額を計算     |

## 🚀 使い方（3ステップ）

### 1️⃣ VS Codeでこのプロジェクトを開く

### 2️⃣ Copilot Chatを起動（`Ctrl+Shift+I`）

### 3️⃣ 質問する

```
@subsidy-consultant 補助金について相談したい
```

あとはAIの質問に答えるだけ！業種・規模・やりたいことを伝えれば、最適な補助金を提案してくれます。

## ⚠️ 注意事項

### 絶対にやってはいけないこと

1. **交付決定前の契約・発注** → 補助対象外
2. **gBizIDの共有・代理申請** → 不正行為
3. **キックバックの受領** → 不正行為
4. **二重受給** → 補助金返還

### 忘れてはいけないこと

1. **実績報告期限の遵守** → 期限超過で補助金不交付
2. **証拠書類の5年間保管** → 立入調査対応
3. **効果報告の実施** → 毎年1回、3〜5年間

# 事業承継・M&A補助金 申請枠と類型

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [事業承継・引継ぎ補助金公式サイト](https://jsh.go.jp/)

---

## 1. 経営革新枠

事業承継・M&A後の経営革新（新事業展開等）に係る費用を支援します。

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| 補助率     | 1/2（一部2/3）                 |
| 補助上限額 | 800万円（一部1,000万円）       |
| 対象経費   | 設備投資、販路開拓費、人件費等 |
| 対象者     | 事業承継・M&Aを行った事業者    |

### 補助率・上限の引上げ条件

| 条件             | 補助率 | 補助上限額 |
| ---------------- | ------ | ---------- |
| 通常             | 1/2    | 800万円    |
| 一定の賃上げ達成 | 2/3    | 1,000万円  |

### 対象経費

| 経費区分 | 例                       |
| -------- | ------------------------ |
| 設備費   | 新事業用の製造機械、什器 |
| 外注費   | 新商品開発の外注         |
| 委託費   | マーケティング調査       |
| 広報費   | 新ブランドのPR           |
| 旅費     | 新規取引先への営業出張   |

### 想定ケース

| パターン      | 事例                                                   |
| ------------- | ------------------------------------------------------ |
| 親族内承継    | 後継者が新業態（カフェ＋雑貨）を展開するための店舗改装 |
| M&A（譲受側） | 買収した企業の製品ラインを拡充するための設備投資       |
| 創業          | 事業承継をきっかけに新サービスを立ち上げ               |

---

## 2. 専門家活用枠

M&Aに係る専門家活用費用を支援します。

| 項目       | 内容                                                    |
| ---------- | ------------------------------------------------------- |
| 補助率     | 1/2〜2/3                                                |
| 補助上限額 | 600万円                                                 |
| 対象経費   | M&A仲介費用、デューデリジェンス費用、弁護士・税理士費用 |
| 対象者     | M&Aを行う譲渡・譲受事業者                               |

### 対象経費の詳細

| 経費                   | 内容                             | 費用目安       |
| ---------------------- | -------------------------------- | -------------- |
| M&A仲介費用            | 仲介会社への手数料               | 100万〜500万円 |
| デューデリジェンス     | 買収先企業の財務・法務・事業調査 | 50万〜300万円  |
| 弁護士費用             | 契約書作成・レビュー             | 30万〜100万円  |
| 税理士・公認会計士費用 | 税務・会計アドバイス             | 30万〜100万円  |
| PMI（経営統合）費用    | M&A後の統合コンサルティング      | 50万〜200万円  |

### 譲渡側・譲受側の違い

| 項目     | 譲渡側                            | 譲受側                           |
| -------- | --------------------------------- | -------------------------------- |
| 補助率   | 2/3                               | 1/2                              |
| 主な経費 | 仲介費用、弁護士費用              | DD費用、PMI費用                  |
| 注意点   | M&A仲介手数料の適正性が審査される | 事業計画の実現可能性が重視される |

---

## 3. 廃業・再チャレンジ枠

事業承継やM&Aに伴う廃業費用を支援します。

| 項目       | 内容                                       |
| ---------- | ------------------------------------------ |
| 補助率     | 1/2〜2/3                                   |
| 補助上限額 | 150万円                                    |
| 対象経費   | 廃業費用（在庫処分、解体費用等）           |
| 対象者     | 廃業を行う事業者、再チャレンジを行う経営者 |

### 対象経費の詳細

| 経費         | 内容                     |
| ------------ | ------------------------ |
| 在庫処分費   | 商品・原材料の処分費用   |
| 解体費       | 店舗・工場の原状回復費用 |
| 廃棄費       | 什器・設備の廃棄費用     |
| リース解約費 | リース契約の中途解約費用 |
| 移転費       | 事業所の移転費用         |

### 想定ケース

| パターン             | 事例                                 |
| -------------------- | ------------------------------------ |
| 廃業→再チャレンジ    | 老舗の飲食店を閉店し、別業態で再創業 |
| M&Aに伴う廃業        | 事業の一部を譲渡し、残りの事業を廃業 |
| 後継者不在による廃業 | 後継者がいないため事業を整理         |

---

## 枠の組み合わせ

### 併用可能な組み合わせ

| 組み合わせ                          | 例                                       |
| ----------------------------------- | ---------------------------------------- |
| 経営革新枠 + 廃業・再チャレンジ枠   | 事業承継で一部事業を廃業し、新事業を開始 |
| 専門家活用枠 + 廃業・再チャレンジ枠 | M&Aで一部事業を譲渡し、残りを廃業        |

> **注意**: 同一経費での併用は不可です。

---

## 枠の選び方フローチャート

```
[事業承継の状況は？]
    │
    ├── 承継後に新事業・経営革新を行いたい
    │   └── 経営革新枠（最大800万〜1,000万円）
    │
    ├── M&Aの仲介・専門家費用を支援してほしい
    │   └── 専門家活用枠（最大600万円）
    │
    ├── 廃業して再チャレンジしたい
    │   └── 廃業・再チャレンジ枠（最大150万円）
    │
    └── 複数に該当する
        └── 組み合わせて申請可能
```

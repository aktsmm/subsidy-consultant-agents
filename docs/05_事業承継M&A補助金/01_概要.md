# 事業承継・M&A補助金 概要

> **情報更新日**: 2026年1月28日  
> **正式名称**: 事業承継・引継ぎ補助金  
> **公式サイト**: [事業承継・引継ぎ補助金](https://jsh.go.jp/)

---

## 制度の仕組み

### 補助金の目的

事業承継やM&Aを契機とした**経営革新等への挑戦**、またM&Aによる**経営資源の引継ぎ**、**廃業・再チャレンジ**を行う中小企業者等を支援する補助金です。

以下の費用を支援することで、円滑な事業承継を促進します：

- 事業承継前の設備投資等にかかる費用
- M&A時の専門家活用にかかる費用
- M&A後のPMI（経営統合作業）にかかる専門家費用や設備投資
- 承継時に伴う廃業にかかる費用

## 3つの申請枠

| 枠                   | 補助率         | 補助上限額               | 主な対象               |
| -------------------- | -------------- | ------------------------ | ---------------------- |
| 経営革新枠           | 1/2（一部2/3） | 800万円（一部1,000万円） | 事業承継後の新事業展開 |
| 専門家活用枠         | 1/2〜2/3       | 600万円                  | M&A時の専門家費用      |
| 廃業・再チャレンジ枠 | 1/2〜2/3       | 150万円                  | 廃業費用               |

---

## 他補助金との比較・併用

### 併用可能なケース

| 補助金              | 対象経費                     |
| ------------------- | ---------------------------- |
| 事業承継・M&A補助金 | 設備投資、専門家費用         |
| IT導入補助金        | ITツール（ソフトウェア）導入 |

> **ポイント**: 同一経費への二重申請は不可ですが、異なる経費であれば併用可能です。

### 事業承継に伴うIT導入の例

```
事業承継を契機にITシステムを刷新する場合：
├── システム更新費用 → IT導入補助金
├── 設備投資 → 事業承継・M&A補助金（経営革新枠）
└── M&A仲介費用 → 事業承継・M&A補助金（専門家活用枠）
```

## コンサルとしての主要チェックポイント

### 初回ヒアリング時

- [ ] 事業承継の種類（親族内承継 / 親族外承継 / M&A）
- [ ] 承継のフェーズ（検討中 / 交渉中 / 完了後）
- [ ] 必要な費用の種類（設備投資 / 専門家 / 廃業費用）
- [ ] gBizIDプライムの取得状況

### 枠の選択

- [ ] 承継後に新事業展開を予定 → **経営革新枠**
- [ ] M&Aの仲介・DD費用が必要 → **専門家活用枠**
- [ ] 廃業・再創業を予定 → **廃業・再チャレンジ枠**

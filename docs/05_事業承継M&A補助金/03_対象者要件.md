# 事業承継・M&A補助金 対象者要件

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [事業承継・引継ぎ補助金公式サイト](https://jsh.go.jp/)

---

## 枠別の対象者

### 経営革新枠

以下のいずれかに該当する中小企業者：

| 対象者     | 条件                                  |
| ---------- | ------------------------------------- |
| 事業承継者 | 親族内・親族外で事業承継を行った者    |
| M&A譲受者  | M&Aにより経営資源を引き継いだ者       |
| 創業者     | 創業後5年以内で、承継的な要素がある者 |

### 専門家活用枠

| 対象者 | 条件                                    |
| ------ | --------------------------------------- |
| 譲渡側 | M&Aにより経営資源を譲り渡す中小企業者   |
| 譲受側 | M&Aにより経営資源を譲り受ける中小企業者 |

### 廃業・再チャレンジ枠

| 対象者         | 条件                               |
| -------------- | ---------------------------------- |
| 廃業者         | 事業承継やM&Aに伴い廃業を行う者    |
| 再チャレンジ者 | 廃業後に別事業で再創業を行う経営者 |

---

## 中小企業の定義

| 業種                     | 資本金        | 従業員数  |
| ------------------------ | ------------- | --------- |
| 製造業、建設業、運輸業等 | 3億円以下     | 300人以下 |
| 卸売業                   | 1億円以下     | 100人以下 |
| サービス業               | 5,000万円以下 | 100人以下 |
| 小売業                   | 5,000万円以下 | 50人以下  |

## 必須準備事項

| 項目           | 経営革新枠 | 専門家活用枠 | 廃業枠 |
| -------------- | :--------: | :----------: | :----: |
| gBizIDプライム |     ✅     |      ✅      |   ✅   |
| 事業計画書     |     ✅     |      -       |   -    |
| M&A関連書類    |   該当時   |      ✅      | 該当時 |
| 廃業計画書     |     -      |      -       |   ✅   |

---

## 対象外となるケース

- [ ] 大企業の子会社（みなし大企業）
- [ ] 直近の公募で同一の補助金を受給中の者
- [ ] 税金の未納がある者
- [ ] M&A仲介等の専門家手数料が著しく高額な場合（専門家活用枠）

---

## 「事業承継」の範囲

### 対象となる事業承継の形態

| 形態       | 説明                       | 経営革新枠 | 専門家活用枠 |
| ---------- | -------------------------- | :--------: | :----------: |
| 親族内承継 | 子や配偶者への承継         |     ✅     |      -       |
| 親族外承継 | 従業員や外部人材への承継   |     ✅     |      -       |
| 株式譲渡   | M&Aによる株式の売買        |     ✅     |      ✅      |
| 事業譲渡   | 事業の一部または全部の譲渡 |     ✅     |      ✅      |
| 合併       | 企業の合併                 |     ✅     |      ✅      |
| 会社分割   | 事業の分割・再編           |     ✅     |      ✅      |

## よくある質問

### Q: まだM&Aが成立していませんが申請できますか？

**A**: 専門家活用枠であれば、M&A検討段階でも申請可能です。経営革新枠は原則、事業承継・M&A完了後の申請となります。

### Q: 個人事業主から法人へ「法人成り」した場合は対象ですか？

**A**: 法人成りのみでは対象外です。経営者の交代を伴う事業承継である必要があります。

### Q: 事業承継・引継ぎ支援センターへの相談は必須ですか？

**A**: 必須ではありませんが、事前相談を行うことで申請書類の質が向上し、採択率が上がる傾向にあります。無料で利用できるため、活用を推奨します。

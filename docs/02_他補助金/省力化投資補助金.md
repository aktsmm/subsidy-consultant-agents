# 省力化投資補助金

> **情報更新日**: 2026年1月28日  
> **正式名称**: 中小企業省力化投資補助金  
> **公式サイト**: [省力化投資補助金](https://shoryokuka.smrj.go.jp/)

---

## 概要

人手不足に悩む中小企業等に対して、**IoT・ロボット等の人手不足解消に効果がある汎用製品**の導入、または**個別の現場や事業内容等に合わせた設備導入・システム構築**等の省力化投資を支援する補助金です。

### 事業目的

中小企業等の**売上拡大や生産性向上**を後押しし、**付加価値額や生産性向上**を図り、**賃上げ**につなげることを目的としています。

---

## 2つの類型

### 1. カタログ注文型

簡易で即効性がある省力化投資を支援します。

| 項目       | 内容                                         |
| ---------- | -------------------------------------------- |
| 対象       | カタログに掲載された省力化効果のある汎用製品 |
| 補助率     | 1/2（賃上げ要件達成で2/3）                   |
| 補助上限額 | 従業員規模により200万円〜1,500万円           |
| 申請方法   | 随時受付中                                   |

#### 補助上限額の詳細

| 従業員数 | 補助上限額（通常） | 補助上限額（賃上げ達成） |
| -------- | ------------------ | ------------------------ |
| 5人以下  | 200万円            | 300万円                  |
| 6〜20人  | 500万円            | 750万円                  |
| 21〜50人 | 1,000万円          | 1,500万円                |
| 51人以上 | 1,000万円          | 1,500万円                |

#### カタログ製品の例

| カテゴリ             | 製品例                     |
| -------------------- | -------------------------- |
| 配膳ロボット         | 飲食店向け自動配膳ロボット |
| 清掃ロボット         | 業務用自動床洗浄機         |
| 検品・仕分けシステム | AI外観検査装置             |
| 無人決済システム     | セルフレジ、自動精算機     |
| 在庫管理システム     | RFID在庫管理               |

### 2. 一般型

事業内容に合わせた多様な設備やシステム導入を支援します。

| 項目       | 内容                                                     |
| ---------- | -------------------------------------------------------- |
| 対象       | 個別現場の設備や事業内容に合わせた設備導入・システム構築 |
| 補助率     | 1/3〜2/3（従業員規模・賃上げ達成により変動）             |
| 補助上限額 | 最大1億円                                                |
| 申請方法   | 公募締切あり                                             |

#### 補助上限額の詳細

| 従業員数 | 補助上限額 |
| -------- | ---------- |
| 5人以下  | 1,000万円  |
| 6〜50人  | 5,000万円  |
| 51人以上 | 1億円      |

## カタログ注文型と一般型の違い

| 項目           | カタログ注文型     | 一般型                 |
| -------------- | ------------------ | ---------------------- |
| 投資内容       | 簡易で即効性がある | オーダーメイド性がある |
| 補助対象       | カタログ掲載製品   | 個別設備・システム     |
| 補助上限       | 最大1,500万円      | 最大1億円              |
| 申請タイミング | 随時               | 公募期間内             |
| 審査難易度     | 比較的容易         | 事業計画審査あり       |

---

## 対象経費

| 経費区分       | 内容                             |
| -------------- | -------------------------------- |
| 設備費         | 省力化製品・設備の購入費         |
| 導入経費       | 設置費、設定費                   |
| システム構築費 | カスタマイズ、連携開発（一般型） |
| 外注費         | 導入コンサルティング             |

---

## 活用例

| ケース  | 業種   | 内容                                                               |
| ------- | ------ | ------------------------------------------------------------------ |
| ケース1 | 製造業 | 部品加工に複合加工機を導入し、人手不足解消と安定した加工精度を実現 |
| ケース2 | 飲食業 | 配膳ロボットを活用し、ホール業務の効率化と顧客満足度向上を両立     |
| ケース3 | 卸売業 | AI技術を活用した自動外観検査装置を導入し、検査工程を効率化         |

## IT導入補助金との違い

| 項目       | 省力化投資補助金     | IT導入補助金          |
| ---------- | -------------------- | --------------------- |
| 対象       | ハードウェア中心     | ソフトウェア中心      |
| 目的       | 人手不足解消・省力化 | 業務効率化・DX        |
| 補助上限額 | 最大1億円            | 最大450万円（通常枠） |
| カタログ型 | あり（随時申請可）   | なし                  |

### 使い分けの目安

| ニーズ                           | 推奨補助金                           |
| -------------------------------- | ------------------------------------ |
| ロボット・自動化機器を導入したい | 省力化投資補助金                     |
| 業務ソフトウェアを導入したい     | IT導入補助金                         |
| 製造設備を導入したい             | 省力化投資補助金 or ものづくり補助金 |
| インボイス対応したい             | IT導入補助金                         |

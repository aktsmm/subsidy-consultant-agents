# 事業承継・M&A補助金

> **情報更新日**: 2026年1月28日  
> **正式名称**: 事業承継・引継ぎ補助金  
> **公式サイト**: [事業承継・引継ぎ補助金](https://jsh.go.jp/)

---

## 概要

事業承継やM&Aを契機とした**経営革新等への挑戦**、またM&Aによる**経営資源の引継ぎ**、**廃業・再チャレンジ**を行う中小企業者等を支援する補助金です。

### 事業目的

- 事業承継前の設備投資等にかかる費用
- M&A時の専門家活用にかかる費用
- M&A後のPMI（経営統合作業）にかかる専門家費用や設備投資
- 承継時に伴う廃業にかかる費用

を支援することで、円滑な事業承継を促進します。

---

## 申請類型

### 1. 経営革新枠

事業承継・M&A後の経営革新（新事業展開等）に係る費用を支援します。

| 項目       | 内容                           |
| ---------- | ------------------------------ |
| 補助率     | 1/2（一部2/3）                 |
| 補助上限額 | 800万円（一部1,000万円）       |
| 対象経費   | 設備投資、販路開拓費、人件費等 |
| 対象者     | 事業承継・M&Aを行った事業者    |

### 2. 専門家活用枠

M&Aに係る専門家活用費用を支援します。

| 項目       | 内容                                                    |
| ---------- | ------------------------------------------------------- |
| 補助率     | 1/2〜2/3                                                |
| 補助上限額 | 600万円                                                 |
| 対象経費   | M&A仲介費用、デューデリジェンス費用、弁護士・税理士費用 |
| 対象者     | M&Aを行う譲渡・譲受事業者                               |

### 3. 廃業・再チャレンジ枠

事業承継やM&Aに伴う廃業費用を支援します。

| 項目       | 内容                                       |
| ---------- | ------------------------------------------ |
| 補助率     | 1/2〜2/3                                   |
| 補助上限額 | 150万円                                    |
| 対象経費   | 廃業費用（在庫処分、解体費用等）           |
| 対象者     | 廃業を行う事業者、再チャレンジを行う経営者 |

---

## 対象となる事業者

### 経営革新枠

以下のいずれかに該当する事業者：

- [ ] 事業承継（親族内・親族外）を行った者
- [ ] M&Aにより経営資源を引き継いだ者
- [ ] 創業（創業後5年以内）を行った者

### 専門家活用枠

- [ ] M&Aにより経営資源を譲り渡す者（譲渡側）
- [ ] M&Aにより経営資源を譲り受ける者（譲受側）

---

## 活用イメージ

| ケース             | 内容                                  |
| ------------------ | ------------------------------------- |
| 親族内承継         | 後継者が新事業展開のため設備投資      |
| M&A（譲受側）      | 買収後のPMI（経営統合）に専門家を活用 |
| M&A（譲渡側）      | 事業譲渡のためのM&A仲介費用           |
| 廃業・再チャレンジ | 廃業後に別事業で再創業                |

---

## IT導入補助金との併用

### 併用可能なケース

| 補助金              | 対象経費                     |
| ------------------- | ---------------------------- |
| 事業承継・M&A補助金 | 設備投資、専門家費用         |
| IT導入補助金        | ITツール（ソフトウェア）導入 |

> **ポイント**: 同一経費への二重申請は不可ですが、異なる経費であれば併用可能です。

### 事業承継に伴うIT導入

事業承継を契機にITシステムを刷新する場合：

- システム更新費用 → IT導入補助金
- 設備投資、専門家活用 → 事業承継・M&A補助金

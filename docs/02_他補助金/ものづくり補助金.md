# ものづくり補助金

> **情報更新日**: 2026年1月28日  
> **正式名称**: ものづくり・商業・サービス生産性向上促進補助金  
> **公式サイト**: [ものづくり補助金総合サイト](https://portal.monodukuri-hojo.jp/)

---

## 概要

中小企業・小規模事業者等が今後複数年にわたり相次いで直面する制度変更（働き方改革、被用者保険の適用拡大、賃上げ、インボイス導入等）等に対応するため、中小企業等が取り組む**革新的な製品・サービスの開発**、**生産プロセス・サービス提供方法の改善**に必要な設備投資等を支援する補助金です。

---

## 申請枠

### 1. 製品・サービス高付加価値化枠

新製品・新サービスの開発の設備投資を支援します。

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| 対象       | 革新的な製品・サービスの開発に必要な設備投資       |
| 補助率     | 中小企業1/2、小規模事業者2/3                       |
| 補助上限額 | 従業員規模により750万円〜4,000万円                 |
| 対象経費   | 機械装置・システム構築費、技術導入費、専門家経費等 |

#### 補助上限額の詳細

| 従業員数 | 補助上限額 |
| -------- | ---------- |
| 5人以下  | 750万円    |
| 6〜20人  | 1,500万円  |
| 21〜50人 | 3,000万円  |
| 51人以上 | 4,000万円  |

### 2. グローバル枠

海外事業に伴う設備・システム投資を支援します。

| 項目       | 内容                                               |
| ---------- | -------------------------------------------------- |
| 対象       | 海外市場への挑戦、インバウンド対応の強化           |
| 補助率     | 中小企業1/2、小規模事業者2/3                       |
| 補助上限額 | 3,000万円                                          |
| 対象経費   | 機械装置・システム構築費、海外旅費、通訳・翻訳費等 |

## 申請要件

### 基本要件

- [ ] 中小企業・小規模事業者であること
- [ ] 付加価値額の年率平均3%以上増加
- [ ] 給与支給総額の年率平均1.5%以上増加
- [ ] 事業場内最低賃金が地域別最低賃金+30円以上

### 必須準備

- [ ] **gBizIDプライム**の取得
- [ ] **jGrants（電子申請システム）**での申請

---

## 審査のポイント

### 加点項目

- [ ] パートナーシップ構築宣言の公開
- [ ] 新規輸出1万者支援プログラムへの登録（グローバル展開加点）
- [ ] 賃上げ加点（給与支給総額の増加）
- [ ] 被災事業者加点

### 事業計画のポイント

```markdown
1. 革新性
   - 他社にない独自の取り組み
   - 地域・業界における新規性

2. 実現可能性
   - 具体的な実施体制
   - 技術的な裏付け

3. 事業効果
   - 付加価値額の増加見込み
   - 生産性向上効果
```

| 項目       | ものづくり補助金     | IT導入補助金                 |

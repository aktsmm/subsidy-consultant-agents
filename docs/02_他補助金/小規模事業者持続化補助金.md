# 小規模事業者持続化補助金

> **情報更新日**: 2026年1月28日  
> **正式名称**: 小規模事業者持続化補助金  
> **公式サイト**:
>
> - [商工会議所地区](https://r6.jizokukahojokin.info/)
> - [商工会地区](https://www.jizokukanb.com/jizokuka_r6h/)

---

## 概要

小規模事業者が「自ら経営計画を策定」し、地域の商工会議所・商工会の支援を受けながら行う、**販路開拓**などの取組を支援する補助金です。

### 事業目的

持続的な経営に向けた経営計画に基づく、小規模事業者等の地道な販路開拓等の取り組みや、業務効率化の取り組みを支援します。

---

## 申請類型

### 1. 一般型（通常枠）

販路開拓や生産性向上の取り組みを支援します。

| 項目       | 内容                                                       |
| ---------- | ---------------------------------------------------------- |
| 補助率     | 2/3                                                        |
| 補助上限額 | 50万円（通常）〜250万円（特別枠）                          |
| 対象経費   | 機械装置等費、広報費、ウェブサイト関連費、展示会等出展費等 |

#### 特別枠（上限額の引き上げ）

| 特別枠       | 補助上限額 | 対象                               |
| ------------ | ---------- | ---------------------------------- |
| 賃金引上げ枠 | 200万円    | 事業場内最低賃金を引き上げる事業者 |
| 卒業枠       | 200万円    | 常時使用する従業員を増やす事業者   |
| 後継者支援枠 | 200万円    | アトツギ甲子園ファイナリスト       |
| 創業枠       | 200万円    | 過去3年以内に創業した事業者        |

### 2. 災害支援枠

自然災害等からの事業再建を支援します。

| 項目       | 内容                              |
| ---------- | --------------------------------- |
| 補助率     | 2/3                               |
| 補助上限額 | 200万円                           |
| 対象       | 令和6年能登半島地震等の被災事業者 |

### 3. 創業型

創業初期の販路開拓や業務改善を支援します。

| 項目       | 内容                 |
| ---------- | -------------------- |
| 補助率     | 2/3                  |
| 補助上限額 | 200万円              |
| 対象       | 創業後間もない事業者 |

### 4. 共同・協業型

複数事業者の展示会、商談会等を活用する販路開拓を支援します。

| 項目       | 内容                                     |
| ---------- | ---------------------------------------- |
| 補助率     | 2/3                                      |
| 補助上限額 | 1,000万円（参加事業者数による）          |
| 対象       | 複数の小規模事業者が連携して行う販路開拓 |

---

## 対象となる小規模事業者

| 業種                                   | 従業員数     |
| -------------------------------------- | ------------ |
| 商業・サービス業（宿泊業・娯楽業除く） | **5人以下**  |
| 宿泊業・娯楽業                         | **20人以下** |
| 製造業その他                           | **20人以下** |

## 申請の特徴

### 商工会議所・商工会への相談が必須

- 経営計画書・補助事業計画書の作成
- 商工会議所・商工会から**「事業支援計画書」**の発行を受ける

### 申請方法

| 地区           | 申請先                       |
| -------------- | ---------------------------- |
| 商工会議所地区 | 商工会議所を経由して事務局へ |
| 商工会地区     | 商工会を経由して事務局へ     |

> **注意**: 管轄地域によって申請先が異なります。

---

## 対象経費

| 経費区分           | 例                                  |
| ------------------ | ----------------------------------- |
| 機械装置等費       | 製造機械、検査機器                  |
| 広報費             | チラシ、パンフレット作成            |
| ウェブサイト関連費 | HP作成・改修（補助金総額の1/4まで） |
| 展示会等出展費     | 展示会・商談会への出展              |
| 旅費               | 販路開拓のための出張費              |
| 開発費             | 新商品の試作開発                    |
| 資料購入費         | 図書、資料                          |
| 借料               | 機器・会場のレンタル                |
| 設備処分費         | 旧設備の処分（補助金総額の1/2まで） |
| 委託・外注費       | デザイン、店舗改装                  |

---

## IT導入補助金との違い

| 項目         | 小規模事業者持続化補助金     | IT導入補助金                 |
| ------------ | ---------------------------- | ---------------------------- |
| 対象者       | 小規模事業者のみ             | 中小企業・小規模事業者       |
| 対象経費     | 販路開拓全般                 | ITツール（ソフトウェア）     |
| 補助上限額   | 最大250万円                  | 最大450万円（通常枠）        |
| 申請プロセス | 商工会議所・商工会の支援必須 | IT導入支援事業者と連携       |
| ウェブサイト | 対象だが上限あり（1/4まで）  | 対象外（業務システムは対象） |

---

## 活用例

| ケース     | 内容                                             |
| ---------- | ------------------------------------------------ |
| 製造業     | 部品加工の受注拡大のため、旋盤機械を導入         |
| 酒造業     | 地域の地酒の販売促進のため、商談会用チラシを制作 |
| 食品加工業 | 新商品の開発・販売のため、フライヤーを購入       |
| 飲食業     | 店舗改装、メニュー刷新のための設備投資           |

# 必要書類チェックリスト

> **情報更新日**: 2026年1月28日  
> **対象補助金**: IT導入補助金を中心に、主要補助金の必要書類をまとめています

---

## IT導入補助金（デジタル化・AI導入補助金）

### 事前準備

| 書類・手続き              | 取得先                                                 | 所要期間 | 備考                             |
| ------------------------- | ------------------------------------------------------ | -------- | -------------------------------- |
| gBizIDプライム            | [GビズID](https://gbiz-id.go.jp/)                      | 約2週間  | 印鑑証明書、登記事項証明書が必要 |
| SECURITY ACTION自己宣言ID | [IPA](https://www.ipa.go.jp/security/security-action/) | 2〜3日   | ★一つ星または★★二つ星            |

### 交付申請時（法人）

| 書類                                   | 取得先   | 有効期限              | 備考                       |
| -------------------------------------- | -------- | --------------------- | -------------------------- |
| 履歴事項全部証明書                     | 法務局   | 発行から**3ヶ月以内** | 登記情報提供サービスでも可 |
| 法人税の納税証明書（その1またはその2） | 税務署   | 直近1期分             | e-Taxでも取得可            |
| 確定申告書（別表一）                   | 手元保管 | 直近1期分             | 税務署受付印または受信通知 |
| 法人事業概況説明書                     | 手元保管 | 直近1期分             | 確定申告時に提出したもの   |
| 決算報告書（貸借対照表・損益計算書）   | 手元保管 | 直近1期分             |                            |

### 交付申請時（個人事業主）

| 書類                                   | 取得先   | 有効期限                   | 備考                       |
| -------------------------------------- | -------- | -------------------------- | -------------------------- |
| 運転免許証 または 住民票               | 市区町村 | 有効期限内 / 発行3ヶ月以内 |                            |
| 所得税の納税証明書（その1またはその2） | 税務署   | 直近1期分                  |                            |
| 確定申告書（第一表）                   | 手元保管 | 直近1期分                  | 税務署受付印または受信通知 |
| 所得税青色申告決算書 または 収支内訳書 | 手元保管 | 直近1期分                  |                            |

### 実績報告時

| 書類                                 | 備考                           |
| ------------------------------------ | ------------------------------ |
| 契約書 または 発注書                 | 交付決定日以降の日付であること |
| 請求書                               | 金額・内訳が明確であること     |
| 領収書 または 振込明細               | 支払日が確認できること         |
| 納品書                               | 納品日が確認できること         |
| ITツール導入画面のスクリーンショット | 利用開始が確認できること       |

| 貸借対照表・損益計算書     | 手元保管           | 直近1期分        |
| 確定申告書（別表一）       | 手元保管           | 直近1期分        |

### 交付申請時（個人事業主）

| 書類                                   | 取得先             | 備考             |
| -------------------------------------- | ------------------ | ---------------- |
| 経営計画書兼補助事業計画書             | 作成               | 様式に従って作成 |
| 事業支援計画書                         | 商工会議所・商工会 | 必須書類         |
| 確定申告書                             | 手元保管           | 直近1期分        |

## 書類取得先一覧

| 書類               | 取得先       | オンライン取得                                                     |
| ------------------ | ------------ | ------------------------------------------------------------------ |
| 履歴事項全部証明書 | 法務局       | [登記情報提供サービス](https://www1.touki.or.jp/)                  |
| 印鑑証明書         | 法務局       | 不可（窓口のみ）                                                   |
| 納税証明書         | 税務署       | [e-Tax](https://www.e-tax.nta.go.jp/)                              |

| gBizIDプライム     | デジタル庁   | [GビズID](https://gbiz-id.go.jp/)                                  |

## 書類準備スケジュール目安

| 書類               | 準備開始時期          | 備考                     |
| ------------------ | --------------------- | ------------------------ |
| gBizIDプライム     | 申請締切の**1ヶ月前** | 取得に約2週間かかる      |
| SECURITY ACTION    | 申請締切の**2週間前** | 2〜3日で取得可能         |
| 履歴事項全部証明書 | 申請締切の**1週間前** | 有効期限3ヶ月に注意      |
| 納税証明書         | 申請締切の**1週間前** | 税務署の混雑に注意       |
| 決算書・確定申告書 | 申請締切の**2週間前** | 手元にあるか確認         |
| 見積書             | 申請締切の**2週間前** | IT導入支援事業者から取得 |

## チェックリストテンプレート

### IT導入補助金 申請準備チェックリスト

```markdown
## IT導入補助金 申請準備チェックリスト

### 顧客情報

- 会社名:
- 申請枠:
- 申請締切:

### 事前準備（申請1ヶ月前）

- [ ] gBizIDプライム取得
- [ ] SECURITY ACTION自己宣言

### 書類準備（申請2週間前）

- [ ] 履歴事項全部証明書（3ヶ月以内）
- [ ] 納税証明書
- [ ] 確定申告書
- [ ] 決算報告書
- [ ] ITツール見積書

### 申請内容確認（申請1週間前）

- [ ] 申請者情報入力完了
- [ ] ITツール情報入力完了
- [ ] 事業計画入力完了
- [ ] 添付書類アップロード完了

### 最終確認（申請前日）

- [ ] 入力内容最終チェック
- [ ] IT導入支援事業者の確認完了
- [ ] 申請提出

### 備考
```

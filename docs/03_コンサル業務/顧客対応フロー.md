# 顧客対応フロー

> **情報更新日**: 2026年1月28日  
> **対象**: IT助成金コンサルティング業務

---

## 顧客対応の全体フロー

```
【フェーズ1: 初回相談】
    ├─ 問い合わせ対応
    ├─ 初回ヒアリング
    └─ 対象者診断・補助金選定
        ↓
【フェーズ2: 提案・契約】
    ├─ 補助金活用提案
    ├─ 見積もり・契約
    └─ IT導入支援事業者との連携
        ↓
【フェーズ3: 申請準備】
    ├─ gBizID取得サポート
    ├─ SECURITY ACTION宣言サポート
    ├─ ITツール選定支援
    └─ 事業計画策定
        ↓
【フェーズ4: 交付申請】
    ├─ 申請書類作成サポート
    ├─ 書類チェック
    └─ 申請提出サポート
        ↓
【フェーズ5: 事業実施】
    ├─ 交付決定後の案内
    ├─ 契約・発注タイミング管理
    └─ 証拠書類管理アドバイス
        ↓
【フェーズ6: 実績報告】
    ├─ 実績報告書作成サポート
    ├─ 書類チェック
    └─ 報告提出サポート
        ↓
【フェーズ7: 効果報告（年次）】
    ├─ リマインド
    ├─ 効果報告書作成サポート
    └─ 継続フォロー
```

### 対象者診断

```
【対象者診断フロー】

1. 業種の確認
   └─ 製造業/卸売業/小売業/サービス業/建設業/その他

2. 規模の確認
   ├─ 資本金: _____ 円
   └─ 従業員数: _____ 人

3. 中小企業に該当するか？
   ├─ 該当 → IT導入補助金、ものづくり補助金等の候補
   └─ 非該当 → 大企業向け施策を検討

4. 小規模事業者に該当するか？
   ├─ 該当 → 小規模事業者持続化補助金も候補
   └─ 非該当 → 中小企業向け補助金を検討

5. 導入内容による補助金選定
   ├─ ITツール（ソフトウェア） → IT導入補助金
   ├─ 設備投資（製造機械等） → ものづくり補助金
   ├─ 省力化機器（ロボット等） → 省力化投資補助金
   └─ 販路開拓（チラシ、展示会等） → 小規模事業者持続化補助金
```

## フェーズ5: 事業実施

### 交付決定後の案内テンプレート

```markdown
## 【重要】交付決定のご連絡と今後の手続きについて

お客様各位

この度、IT導入補助金の交付決定がおりましたことをお知らせいたします。

### 今後の手続き

1. **契約・発注**
   - 交付決定日以降に契約・発注を行ってください
   - 交付決定前の契約・発注は補助対象外となります

2. **事業実施**
   - 実施期間: ○年○月○日 ～ ○年○月○日
   - ITツールの導入・運用開始

3. **証拠書類の保管**
   以下の書類を必ず保管してください：
   - 契約書（発注書）
   - 請求書
   - 領収書（振込明細）
   - 納品書
   - 導入画面のスクリーンショット

4. **実績報告**
   - 報告期限: ○年○月○日
   - 上記の証拠書類をもとに報告

### ご不明点

ご不明な点がございましたら、お気軽にお問い合わせください。
```

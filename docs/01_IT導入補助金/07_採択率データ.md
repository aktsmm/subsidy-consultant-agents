## 採択率を上げるポイント

### 審査項目（一般的な傾向）

| 審査項目                 | 配点イメージ | ポイント                   |
| ------------------------ | ------------ | -------------------------- |
| 事業計画の妥当性         | 高           | 具体的な数値目標、現状分析 |
| 生産性向上効果           | 高           | 労働生産性の向上見込み     |
| 導入するITツールの適切性 | 中           | 課題解決に適したツール選定 |
| 事業実施体制             | 中           | IT導入支援事業者との連携   |
| 賃上げへの取組           | 加点         | 賃上げ計画の明示           |

### 加点項目（2025年度参考）

- [ ] **賃上げ加点**: 給与支給総額の年率1.5%以上増加
- [ ] **インボイス対応加点**: 免税事業者からインボイス発行事業者に転換
- [ ] **パートナーシップ構築宣言**: 宣言を公開している
- [ ] **導入するITツールとしてクラウド製品を選定**
- [ ] **セキュリティ対策の実施**

### 不採択になりやすいケース

- [ ] 事業計画が抽象的・具体性に欠ける
- [ ] 導入効果（生産性向上）が不明確
- [ ] 既存システムとの重複
- [ ] 申請書類の不備・矛盾
- [ ] 過去に補助金で問題があった

## 申請枠別の戦略

### 通常枠（採択率50%前後）

**競争率が高いため、事業計画の質が重要**

```markdown
## 事業計画のポイント

1. 現状分析
   - 具体的な課題の明示
   - 数値データによる裏付け

2. 導入効果
   - 定量的な目標設定（売上○%増、作業時間○時間削減）
   - 労働生産性向上の計算根拠

3. 実施体制
   - IT導入支援事業者との役割分担
   - 社内の推進体制
```

### インボイス枠（採択率55〜60%）

**申請数が多いが、採択率はやや高め**

- インボイス制度への対応が明確であること
- 小規模事業者は補助率が高いため有利

### セキュリティ対策推進枠（採択率90%以上）

**申請数が少なく、採択率が高い穴場**

- 「サイバーセキュリティお助け隊サービスリスト」掲載サービスであること
- セキュリティ対策の必要性を明確に説明

## コンサルとしての活用ポイント

### 顧客への説明

```markdown
## 採択率に関する説明テンプレート

【現状の採択率】

- 通常枠: 約50%（2件に1件が採択）
- インボイス枠: 約55〜60%
- セキュリティ対策推進枠: 約90%以上

【採択されるためのポイント】

1. 事業計画の具体性・説得力
2. 導入効果の数値化
3. 申請書類の正確性・完全性
4. 加点項目の活用

【コンサルとしてのサポート】

- 事業計画書のレビュー・アドバイス
- 導入効果の試算サポート
- 申請書類のチェック
- 加点項目の確認
```

### 申請枠の選定アドバイス

| 顧客の状況             | 推奨申請枠             | 理由                           |
| ---------------------- | ---------------------- | ------------------------------ |
| インボイス対応が必要   | インボイス枠           | 補助率が高く、採択率もやや高い |
| セキュリティ強化が課題 | セキュリティ対策推進枠 | 採択率が非常に高い             |
| 業務効率化したい       | 通常枠                 | 幅広いITツールが対象           |
| 複数社で連携したい     | 複数者連携枠           | 補助上限額が大きい             |

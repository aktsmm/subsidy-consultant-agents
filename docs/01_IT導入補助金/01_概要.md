# IT導入補助金（デジタル化・AI導入補助金）概要

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [IT導入補助金公式サイト](https://it-shien.smrj.go.jp/)

## 重要: 2026年度の名称変更

2026年度より「**IT導入補助金**」は「**デジタル化・AI導入補助金**」に名称変更されました。

- IT導入にとどまらず、**AI導入支援**や**生産性向上**を強化する方針
- 事務局運営: TOPPAN株式会社（独立行政法人中小企業基盤整備機構より採択）

---

## 制度の仕組み

### 補助金の目的

中小企業・小規模事業者等の**労働生産性の向上**を目的として、業務効率化やDX等に向けたITツール（ソフトウェア、サービス等）の導入を支援する補助金です。

### 3者の関係

```
┌─────────────────────────────────────────────────────────────┐
│                    事務局（TOPPAN）                          │
│  ・ITツール・IT導入支援事業者の審査・登録                      │
│  ・交付申請の審査・交付決定                                   │
│  ・実績報告の確認・補助金交付                                 │
└─────────────────────────────────────────────────────────────┘
          ▲                              ▲
          │ 登録申請                      │ 交付申請
          │                              │
┌─────────────────────┐          ┌─────────────────────┐
│ IT導入支援事業者      │◀────────▶│ 中小企業・小規模事業者 │
│ （ITベンダー）        │  連携     │ （補助事業者）        │
│                      │          │                      │
│ ・ITツールの提供      │          │ ・ITツールの導入      │
│ ・導入サポート        │          │ ・補助金申請          │
│ ・アフターフォロー    │          │ ・効果報告            │
└─────────────────────┘          └─────────────────────┘
```

### 補助対象となるITツール

事前に事務局の審査を受け、補助金HPに公開（登録）されているITツールが対象です。

| 対象経費                   | 例                                           |
| -------------------------- | -------------------------------------------- |
| ソフトウェア購入費         | 会計ソフト、受発注システム、在庫管理システム |
| クラウドサービス利用料     | SaaS利用料（最大2年分）                      |
| 導入コンサルティング費用   | 導入支援、設定代行                           |
| 導入研修費用               | 操作研修、マニュアル作成                     |
| ハードウェア（一部枠のみ） | PC、タブレット、レジ等                       |

## 申請枠の種類（5類型）

| 申請枠                                 | 概要                                                   | 主な対象ツール               |
| -------------------------------------- | ------------------------------------------------------ | ---------------------------- |
| **通常枠**                             | 事業のデジタル化を目的としたソフトウェア・システム導入 | 在庫管理、決済ソフト、CRM    |
| **インボイス枠（インボイス対応類型）** | インボイス制度対応のソフト・ハードウェア導入           | 会計ソフト、受発注ソフト、PC |
| **インボイス枠（電子取引類型）**       | 商流単位での受発注システム導入                         | 受発注システム（発注側負担） |
| **セキュリティ対策推進枠**             | サイバー攻撃対策・リスク低減策                         | ネットワーク監視、UTM        |
| **複数者連携デジタル化・AI導入枠**     | 複数事業者連携による地域DX推進                         | データ分析システム           |

→ 詳細は [02\_申請枠と類型.md](02_申請枠と類型.md) を参照

## 申請の流れ（概要）

```
STEP1: 本事業の理解（公募要領確認）
    ↓
STEP2: gBizID取得・SECURITY ACTION宣言
    ↓
STEP3: IT導入支援事業者・ITツールの選定
    ↓
STEP4: 交付申請（IT導入支援事業者と共同作成）
    ↓
STEP5: 交付決定
    ↓
STEP6: ITツール導入・事業実施
    ↓
STEP7: 実績報告
    ↓
STEP8: 補助金交付
    ↓
STEP9: 効果報告（年次）
```

→ 詳細は [04\_申請フロー.md](04_申請フロー.md) を参照

## コンサルとしての主要チェックポイント

### 初回ヒアリング時

- [ ] 業種・従業員数・資本金の確認（対象者要件）
- [ ] 導入したいITツールの明確化
- [ ] gBizIDプライムの取得状況
- [ ] SECURITY ACTION自己宣言の実施状況
- [ ] 過去のIT導入補助金利用履歴

### 申請準備時

- [ ] IT導入支援事業者の選定
- [ ] ITツールの選定（カタログから）
- [ ] 事業計画の策定
- [ ] 必要書類の準備

### 交付決定後

- [ ] 契約・発注のタイミング管理
- [ ] 実績報告期限の把握
- [ ] 効果報告（年次）のスケジュール管理

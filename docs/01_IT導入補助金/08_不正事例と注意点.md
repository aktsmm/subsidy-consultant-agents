# IT導入補助金 不正事例と注意点

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [IT導入補助金は不正を絶対に許しません](https://it-shien.smrj.go.jp/antifraud/)

---

## 重要メッセージ

> **IT導入補助金は不正を絶対に許しません**
>
> 事務局では、本補助金の交付規程の定めに則り、IT導入支援事業者、補助事業者に対し、現地確認を含めた立入調査を行っております。

---

## 不正行為の類型

### 1. ITツール実質無償提供

| 概要   | 補助事業者がITツールを購入するが、後日IT導入支援事業者等から購入費用が返金される       |
| ------ | -------------------------------------------------------------------------------------- |
| 具体例 | 100万円のソフトウェアを購入後、「紹介料」「謝礼」等の名目で100万円がキックバックされる |
| 問題点 | 実質的に自己負担がなく、補助金を不正に受給している                                     |

### 2. キックバック

| 概要   | 補助事業者がIT導入支援事業者等から金銭を受け取る                   |
| ------ | ------------------------------------------------------------------ |
| 具体例 | 「紹介料」「コンサルティング料」「協力金」等の名目で現金を受け取る |
| 問題点 | 補助金の一部が補助事業者に還流している                             |

### 3. 代理申請

| 概要   | 補助事業者がgBizID等を他者に共有し、申請手続きを代行させる |
| ------ | ---------------------------------------------------------- |
| 具体例 | IT導入支援事業者にgBizIDのID・パスワードを渡して申請させる |
| 問題点 | 本人確認の趣旨に反し、なりすまし申請の温床となる           |

### 4. 未導入・未実施

| 概要   | ITツールが実際に導入されていない、役務が遂行されていない           |
| ------ | ------------------------------------------------------------------ |
| 具体例 | 契約書・請求書は存在するが、実際にはソフトウェアが利用されていない |
| 問題点 | 補助金の目的（生産性向上）が達成されていない                       |

### 5. 二重受給

| 概要   | 同じ経費で他の補助金も受給している                         |
| ------ | ---------------------------------------------------------- |
| 具体例 | IT導入補助金とものづくり補助金で同じシステム導入費用を申請 |
| 問題点 | 補助金の重複受給は禁止されている                           |

### 6. 企業実態偽装

| 概要   | 従業員数・資本金等を偽って申請する                                      |
| ------ | ----------------------------------------------------------------------- |
| 具体例 | 実際は従業員100人だが、50人以下と申告して小規模事業者の高い補助率を適用 |
| 問題点 | 補助対象者要件を偽っている                                              |

---

## 不正行為への処分

### 補助事業者への処分

| 処分             | 内容                           |
| ---------------- | ------------------------------ |
| **交付決定取消** | 補助金を受ける権利の喪失       |
| **補助金返還**   | 受領済みの補助金全額の返還     |
| **加算金**       | 返還額に対する年10.95%の加算金 |
| **延滞金**       | 返還期限後の延滞金             |
| **事業者名公表** | 不正事業者として公表           |

### IT導入支援事業者への処分

| 処分                   | 内容                                                 |
| ---------------------- | ---------------------------------------------------- |
| **登録取消**           | IT導入支援事業者としての登録抹消                     |
| **全事業での登録取消** | 中小機構が所管する全てのIT導入補助金事業での登録取消 |
| **事業者名公表**       | 登録取消事業者として公表                             |
| **警察への通報**       | 刑事告発の可能性                                     |

## 登録取消となったIT導入支援事業者

### 登録取消事業者一覧

[IT導入支援事業者の登録取消について](https://it-shien.smrj.go.jp/pdf/deregistration_list.pdf)

### 登録取消事業者数（2025年度時点）

- 累計で**数十社**が登録取消
- 主に**キックバック**や**ITツール実質無償提供**が理由

### コンサルとしての確認ポイント

- IT導入支援事業者を選定する際は、登録取消リストに含まれていないか確認
- 過去に登録取消を受けた代表者が関与する新会社にも注意

## 自主返還の手続き

不正に関与した場合、自主的に補助金を返還する手続きがあります。

### 返還のケース

| ケース               | 提出書類   | 加算金   |
| -------------------- | ---------- | -------- |
| 不正を知りながら関与 | 自己申告書 | **あり** |
| 不正を知らずに関与   | 誓約書     | **なし** |

### 提出書類

- [自己申告書（不正を知りながら関与した場合）](https://it-shien.smrj.go.jp/pdf/it_jikoshinkoku.docx)
- [誓約書（不正を知らずに関与した場合）](https://it-shien.smrj.go.jp/pdf/it_seiyaku.docx)

### 提出先

[自主返還提出フォーム](https://it-shien.ent.box.com/f/a01cbc5730f24e9b896a0462c7351eec)

## コンサルとしての注意点

### 顧客への説明事項

```markdown
## 不正行為に関する説明テンプレート

【重要】IT導入補助金の不正行為について

IT導入補助金では、以下の行為は不正とみなされ、厳しい処分を受けます。

❌ やってはいけないこと

- IT導入支援事業者から現金や商品券をもらう
- gBizIDのID・パスワードを他人に渡す
- 実際に使わないITツールを申請する
- 他の補助金と同じ経費を二重申請する
- 従業員数や資本金を偽る

⚠️ 処分

- 補助金の全額返還（加算金・延滞金を含む）
- 事業者名の公表
- 今後の補助金申請への影響
- 刑事告発の可能性

✅ 正しい補助金の利用

- 実際に使うITツールを申請する
- 自分でgBizIDにログインして申請する
- 証拠書類をきちんと保管する
- 効果報告を正直に行う
```

### IT導入支援事業者への確認事項

- [ ] 登録取消リストに含まれていないか
- [ ] 「紹介料」「キックバック」等の話を持ちかけてこないか
- [ ] gBizIDのログイン情報を求めてこないか
- [ ] 見積もりや契約内容が不自然ではないか

### 不正の兆候

以下のような話があった場合は要注意：

- 「補助金が出たら一部をお返しします」
- 「申請は全部こちらでやりますので、IDを教えてください」
- 「とりあえず申請して、使わなかったら解約すればいい」
- 「他のお客様もこうしていますから大丈夫」

## 立入調査について

事務局は補助事業者・IT導入支援事業者に対して、現地確認を含めた立入調査を行う権限があります。

### 調査内容

- ITツールの導入状況確認
- 証拠書類の確認
- 実際の利用状況のヒアリング
- 支払い状況の確認

### 調査への対応

- 正当な理由なく調査を拒否した場合も処分の対象
- 証拠書類は補助事業完了後5年間保管義務

---

## ITツール継続利用状況調査

事務局では、ITツールの利用状況に関する定期調査を実施しています。

### 調査対象

- 補助金交付を受けた全ての補助事業者

### 調査内容

- ITツールの継続利用状況
- 解約・利用中止の有無
- 事業廃止・事業承継の有無

### 注意事項

- ITツールを解約・利用中止した場合は、事務局への**辞退届の提出が必要**
- 正しく報告しない場合、不正とみなされる可能性

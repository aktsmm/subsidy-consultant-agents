# IT導入補助金 申請フロー

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [新規申請・手続きフロー詳細](https://it-shien.smrj.go.jp/applicant/flow/)

---

## 申請フロー全体像

```
【事前準備フェーズ】
    STEP1: 本事業の理解
        ↓
    STEP2: gBizID取得・SECURITY ACTION宣言
        ↓
    STEP3: IT導入支援事業者・ITツールの選定
        ↓
【申請フェーズ】
    STEP4: 交付申請
        ↓
    STEP5: 交付決定 ← ★ここまで契約・発注してはいけない
        ↓
【事業実施フェーズ】
    STEP6: ITツール導入・事業実施
        ↓
    STEP7: 実績報告
        ↓
    STEP8: 補助金交付
        ↓
【事後フェーズ】
    STEP9: 効果報告（年次）
```

---

## 各ステップの詳細

### STEP1: 本事業の理解

| 項目     | 内容                           |
| -------- | ------------------------------ |
| 所要時間 | 1〜2時間                       |
| 実施者   | 中小企業（コンサルがサポート） |

#### 確認事項

- [ ] 公募要領を確認
- [ ] 交付規程を確認
- [ ] 自社が対象者要件を満たすか確認
- [ ] 申請したい枠・類型を決定
- [ ] スケジュールを確認

#### 参照資料

- [公募要領](https://it-shien.smrj.go.jp/download/)
- [交付規程](https://it-shien.smrj.go.jp/download/)

---

### STEP2: gBizID取得・SECURITY ACTION宣言

| 項目     | 内容                                     |
| -------- | ---------------------------------------- |
| 所要時間 | gBizID: 約2週間、SECURITY ACTION: 2〜3日 |
| 実施者   | 中小企業                                 |

#### gBizIDプライムの取得手順

1. [GビズID](https://gbiz-id.go.jp/top/)にアクセス
2. 「gBizIDプライム作成」を選択
3. 必要情報を入力（法人番号、代表者情報など）
4. 必要書類を郵送（印鑑証明書、登記事項証明書）
5. 審査完了後、アカウント発行（約2週間）

#### SECURITY ACTION自己宣言の手順

1. [SECURITY ACTION](https://www.ipa.go.jp/security/security-action/)にアクセス
2. 「自己宣言する」を選択
3. 宣言レベルを選択（★一つ星 or ★★二つ星）
4. 必要事項を入力
5. 自己宣言IDを取得（即日〜2〜3日）

> **重要**: 交付申請時に「自己宣言ID」の入力が必要です。

---

### STEP3: IT導入支援事業者・ITツールの選定

| 項目     | 内容                               |
| -------- | ---------------------------------- |
| 所要時間 | 1週間〜1ヶ月                       |
| 実施者   | 中小企業（IT導入支援事業者と相談） |

#### IT導入支援事業者の探し方

1. [ITツール・IT導入支援事業者検索](https://it-shien.smrj.go.jp/search/)にアクセス
2. 業種・地域・導入したいツールで検索
3. 複数の事業者に相談・見積もり取得
4. サポート内容・実績を比較して決定

#### ITツールの選定ポイント

- [ ] 自社の課題を解決できるか
- [ ] 補助対象となるツールか（カタログ掲載されているか）
- [ ] 導入後のサポート体制
- [ ] 価格・ランニングコスト
- [ ] 他システムとの連携

---

### STEP4: 交付申請

| 項目     | 内容                        |
| -------- | --------------------------- |
| 所要時間 | 1〜2週間                    |
| 実施者   | 中小企業 + IT導入支援事業者 |

#### 交付申請の手順

1. IT導入支援事業者から「申請マイページ」の招待を受ける
2. 申請マイページにログイン（gBizIDを使用）
3. 申請者基本情報を入力
   - 代表者氏名
   - 法人番号
   - 連絡先
   - 決算情報
4. 交付申請に必要な情報を入力
   - 導入するITツール
   - 事業計画
   - 補助金申請額
5. 必要書類を添付
6. IT導入支援事業者がITツール情報・事業計画値を入力
7. 申請内容を最終確認
8. 宣誓を行い、事務局へ提出

#### 必要書類（法人の場合）

- [ ] 履歴事項全部証明書（発行から3ヶ月以内）
- [ ] 法人税の納税証明書（その1またはその2）
- [ ] 確定申告書（直近1期分）
- [ ] 決算報告書（直近1期分）

#### 必要書類（個人事業主の場合）

- [ ] 運転免許証または住民票
- [ ] 所得税の納税証明書（その1またはその2）
- [ ] 確定申告書（直近1期分）

---

### STEP5: 交付決定

| 項目     | 内容                |
| -------- | ------------------- |
| 所要時間 | 申請締切から約1ヶ月 |
| 実施者   | 事務局              |

#### 交付決定の通知

- 申請マイページで結果を確認
- 交付決定通知書のダウンロード

> **重要**: 交付決定**前**に契約・発注・支払いを行った場合、補助対象外となります。

---

### STEP6: ITツール導入・事業実施

| 項目     | 内容                           |
| -------- | ------------------------------ |
| 所要時間 | 交付決定日〜事業実施期間終了日 |
| 実施者   | 中小企業 + IT導入支援事業者    |

#### 事業実施のポイント

- [ ] 交付決定後に契約・発注を行う
- [ ] ITツールの導入・設定
- [ ] 運用開始
- [ ] 証拠書類の保管（契約書、請求書、領収書、納品書等）

#### 証拠書類の例

- 契約書または発注書
- 請求書
- 領収書または振込明細
- 納品書
- ITツールの導入画面スクリーンショット

---

### STEP7: 実績報告

| 項目     | 内容                         |
| -------- | ---------------------------- |
| 所要時間 | 事業実施完了後〜実績報告期限 |
| 実施者   | 中小企業 + IT導入支援事業者  |

#### 実績報告の手順

1. 申請マイページにログイン
2. 実績報告情報を入力
   - 実際の導入日
   - 支払金額
   - 導入状況
3. 証拠書類を添付
4. IT導入支援事業者が確認・承認
5. 事務局へ提出

> **注意**: 実績報告期限を過ぎると補助金が交付されません。

---

### STEP8: 補助金交付

| 項目     | 内容                    |
| -------- | ----------------------- |
| 所要時間 | 実績報告承認後〜約1ヶ月 |
| 実施者   | 事務局                  |

#### 交付の流れ

1. 事務局が実績報告を審査
2. 補助金確定通知
3. 指定口座に補助金振込

---

### STEP9: 効果報告（年次）

| 項目     | 内容                        |
| -------- | --------------------------- |
| 所要時間 | 毎年1回（3年間または5年間） |
| 実施者   | 中小企業                    |

#### 効果報告の内容

- 売上高の変化
- 従業員数の変化
- 労働生産性の変化
- ITツールの継続利用状況

> **注意**: 効果報告を行わない場合、次回以降のIT導入補助金申請に影響があります。

---

## タイムライン例（2026年度想定）

```
2026年2月    gBizID取得開始、SECURITY ACTION宣言
     ↓
2026年3月    公募要領公開、IT導入支援事業者選定
     ↓
2026年3月30日 交付申請受付開始
     ↓
2026年4月    交付申請（1次締切を想定）
     ↓
2026年5月    交付決定（1次締切分）
     ↓
2026年5月〜   ITツール導入・事業実施
     ↓
2026年10月頃  実績報告（締切は交付決定日により異なる）
     ↓
2026年11月〜  補助金交付
     ↓
2027年〜     効果報告（年次）
```

## コンサルとしての支援ポイント

### 各フェーズでの支援内容

| フェーズ | 支援内容                                               |
| -------- | ------------------------------------------------------ |
| 事前準備 | 対象者診断、gBizID取得サポート、ITツール選定アドバイス |
| 申請     | 事業計画策定、必要書類準備、申請書類チェック           |
| 事業実施 | 進捗管理、証拠書類管理アドバイス                       |
| 実績報告 | 報告書作成サポート、書類チェック                       |
| 効果報告 | リマインド、報告書作成サポート                         |

### よくあるミス・注意点

1. **交付決定前の契約・発注** → 補助対象外
2. **実績報告期限の失念** → 補助金不交付
3. **証拠書類の不備** → 補助金減額
4. **効果報告の未実施** → 次回申請に影響

# IT導入補助金 対象者要件

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [申請の対象となる方](https://it-shien.smrj.go.jp/applicant/subsidy/)

## 中小企業の定義

日本国内で法人登記され、日本国内で事業を営む法人または個人が対象です。

### 業種別の資本金・従業員基準

| No  | 業種分類                                                       | 資本金        | 従業員    |
| --- | -------------------------------------------------------------- | ------------- | --------- |
| 1   | 製造業、建設業、運輸業                                         | 3億円以下     | 300人以下 |
| 2   | 卸売業                                                         | 1億円以下     | 100人以下 |
| 3   | サービス業（ソフトウェア業、情報処理サービス業、旅館業を除く） | 5,000万円以下 | 100人以下 |
| 4   | 小売業                                                         | 5,000万円以下 | 50人以下  |
| 5   | ゴム製品製造業（一部除く）                                     | 3億円以下     | 900人以下 |
| 6   | ソフトウェア業または情報処理サービス業                         | 3億円以下     | 300人以下 |
| 7   | 旅館業                                                         | 5,000万円以下 | 200人以下 |
| 8   | その他の業種                                                   | 3億円以下     | 300人以下 |

> **判定方法**: 資本金**または**従業員数のいずれかを満たせばOK

## 小規模事業者の定義

| No  | 業種                                     | 従業員       |
| --- | ---------------------------------------- | ------------ |
| 1   | 商業・サービス業（宿泊業・娯楽業を除く） | **5人以下**  |
| 2   | サービス業のうち宿泊業・娯楽業           | **20人以下** |
| 3   | 製造業その他                             | **20人以下** |

> **注意**: 業種分類9～15の組織形態は小規模事業者に該当しません。

### 「常時使用する従業員」の定義

労働基準法第20条に規定する「あらかじめ解雇の予告を必要とする者」を指します。

**含まれない者**:

- 日雇い労働者
- 2ヶ月以内の期間を定めて使用される者
- 季節的業務に4ヶ月以内の期間を定めて使用される者
- 試用期間中の者

## 申請要件チェックリスト

### 必須要件

- [ ] **gBizIDプライム**を取得済み
- [ ] **SECURITY ACTION**の自己宣言を実施済み（★一つ星または★★二つ星）
- [ ] 日本国内で法人登記されている（または個人事業主）
- [ ] 日本国内で事業を営んでいる

### gBizIDプライムの取得

| 項目       | 内容                                     |
| ---------- | ---------------------------------------- |
| 取得サイト | [GビズID](https://gbiz-id.go.jp/top/)    |
| 取得期間   | 約2週間（書類郵送の場合）                |
| 必要書類   | 印鑑証明書、登記事項証明書（法人の場合） |
| 費用       | 無料                                     |

> **注意**: 申請締切直前の取得は間に合わない可能性があります。早めの取得を推奨。

### SECURITY ACTION自己宣言

| 項目       | 内容                                                               |
| ---------- | ------------------------------------------------------------------ |
| 申請サイト | [SECURITY ACTION](https://www.ipa.go.jp/security/security-action/) |
| 宣言レベル | ★一つ星（最低限）または★★二つ星                                    |
| 取得期間   | 約2〜3日                                                           |
| 費用       | 無料                                                               |

#### 一つ星の宣言内容

「情報セキュリティ5か条」に取り組むことを宣言：

1. OSやソフトウェアは常に最新に
2. ウイルス対策ソフトを導入
3. パスワードを強化
4. 共有設定を見直し
5. 脅威や攻撃の手口を知る

## 対象外となるケース

以下に該当する場合は申請できません：

### 事業者に関する要件

- [ ] 発行済株式の総数の1/2以上を同一の大企業が所有
- [ ] 発行済株式の総数の2/3以上を大企業が所有
- [ ] 大企業の役員が代表者を兼ねている
- [ ] 大企業の役員が役員総数の1/2以上を占めている
- [ ] 風俗営業等（一部例外あり）
- [ ] 反社会的勢力との関係がある

### 過去の補助金に関する要件

- [ ] 過年度IT導入補助金で不正が確認された
- [ ] IT導入支援事業者として登録取消を受けた
- [ ] 他の補助金で不正が確認された

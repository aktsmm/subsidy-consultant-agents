# IT導入補助金 申請枠と類型

> **情報更新日**: 2026年1月28日  
> **注意**: 2026年度の公募要領は準備中のため、以下は2025年度情報を参考として記載しています。

---

## 申請枠一覧

### 1. 通常枠

事業のデジタル化を目的としたソフトウェアやシステムの導入を支援します。

| 項目             | 内容                                           |
| ---------------- | ---------------------------------------------- |
| **補助対象**     | ソフトウェア購入費、クラウド利用料、導入関連費 |
| **補助率**       | 1/2以内                                        |
| **補助上限額**   | 150万円未満 ～ 450万円以下                     |
| **対象ツール例** | 在庫管理システム、決済ソフト、CRM、ERP         |

#### 対象となるプロセス

以下の業務プロセスを1つ以上含むITツールが対象：

- 顧客対応・販売支援
- 決済・債権債務・資金回収
- 供給・在庫・物流
- 会計・財務・経営
- 総務・人事・給与・労務
- 業種固有プロセス

---

### 2. インボイス枠（インボイス対応類型）

インボイス制度に対応した会計ソフト、受発注ソフト、決済ソフト、PC・ハードウェア等の導入を支援します。

| 項目             | 内容                                                                  |
| ---------------- | --------------------------------------------------------------------- |
| **補助対象**     | インボイス対応ソフト、PC・タブレット・レジ等のハードウェア            |
| **補助率**       | 中小企業: 2/3以内、小規模事業者: 3/4以内（一部4/5）                   |
| **補助上限額**   | ソフト: 350万円以下、ハード: 20万円以下（PC等）/ 20万円以下（レジ等） |
| **対象ツール例** | 会計ソフト、受発注ソフト、決済ソフト、POSレジ、PC、タブレット         |

#### 特徴

- **ハードウェアも対象**（他の枠ではソフトウェアのみ）
- 小規模事業者は**補助率が高い**
- インボイス制度への対応が必須要件

---

### 3. インボイス枠（電子取引類型）

インボイス制度に対応した受発注システムを**商流単位**で導入する企業を支援します。

| 項目             | 内容                                         |
| ---------------- | -------------------------------------------- |
| **補助対象**     | 受発注システム（発注側が費用負担するケース） |
| **補助率**       | 中小企業: 2/3以内、大企業: 1/2以内           |
| **補助上限額**   | 取引先1社あたり350万円                       |
| **対象ツール例** | EDI、受発注プラットフォーム                  |

#### 特徴

- **大企業も申請可能**（発注側として）
- 取引先（中小企業）への導入を発注側が負担
- 商流全体のデジタル化を促進

---

### 4. セキュリティ対策推進枠

サイバー攻撃の増加に伴う潜在的なリスクに対処するため、サイバーインシデントに関する様々なリスク低減策を支援します。

| 項目             | 内容                                                         |
| ---------------- | ------------------------------------------------------------ |
| **補助対象**     | 「サイバーセキュリティお助け隊サービスリスト」掲載のサービス |
| **補助率**       | 1/2以内                                                      |
| **補助上限額**   | 100万円                                                      |
| **補助対象期間** | 最大2年分                                                    |
| **対象ツール例** | UTM、EDR、ネットワーク監視、セキュリティ診断                 |

#### 要件

- IPAが公表する「サイバーセキュリティお助け隊サービスリスト」に掲載されているサービスであること
- [サービスリスト](https://www.ipa.go.jp/security/otasuketai-pr/)

---

### 5. 複数者連携デジタル化・AI導入枠

複数の中小企業・小規模事業者等が連携して地域DXの実現や、生産性の向上を図る取り組みを支援します。

| 項目             | 内容                                             |
| ---------------- | ------------------------------------------------ |
| **補助対象**     | 複数事業者で共同利用するシステム・サービス       |
| **補助率**       | 1/2以内 ～ 2/3以内                               |
| **補助上限額**   | 3,000万円（事業費）+ 200万円（事務費・専門家費） |
| **対象ツール例** | 地域データ分析、共同受発注システム、観光DX       |

#### 要件

- 10者以上の中小企業・小規模事業者等が参画
- 商工会議所・商工会等が幹事社として申請
- 地域全体のDX推進を目的とすること

---

## 申請枠の選び方フローチャート

```
導入したいITツールは？
│
├─ インボイス対応の会計・受発注ソフト
│   │
│   ├─ 自社で使う ──────────────→ インボイス枠（インボイス対応類型）
│   │
│   └─ 取引先に導入させたい ────→ インボイス枠（電子取引類型）
│
├─ セキュリティ対策サービス ────→ セキュリティ対策推進枠
│
├─ 地域連携・複数社での共同導入 ──→ 複数者連携デジタル化・AI導入枠
│
└─ その他のソフトウェア・システム ──→ 通常枠
```

## 補助率・補助上限額 比較表

| 申請枠                             | 補助率   | 補助上限額                   |
| ---------------------------------- | -------- | ---------------------------- |
| 通常枠                             | 1/2      | 150万円～450万円             |
| インボイス枠（インボイス対応類型） | 2/3～4/5 | ソフト350万円 + ハード20万円 |
| インボイス枠（電子取引類型）       | 1/2～2/3 | 350万円/社                   |
| セキュリティ対策推進枠             | 1/2      | 100万円                      |
| 複数者連携デジタル化・AI導入枠     | 1/2～2/3 | 3,000万円 + 200万円          |

## 2026年度の変更点（予定）

公募要領公開（2026年3月下旬予定）後に確認が必要な項目：

- [ ] 各枠の補助率・補助上限額の変更有無
- [ ] AI導入に関する新たな要件・加点項目
- [ ] 賃上げ要件の強化有無
- [ ] 対象経費の拡大・縮小

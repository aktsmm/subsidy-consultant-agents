# IT導入補助金 IT導入支援事業者

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [ITベンダー・サービス事業者のみなさま](https://it-shien.smrj.go.jp/itvendor/)

---

## IT導入支援事業者とは

IT導入支援事業者とは、生産性向上を目指す中小企業・小規模事業者等に対してITツールを導入し、補助事業を円滑に遂行するための支援を行う事業者です。

### IT導入支援事業者の役割

| 役割             | 内容                                 |
| ---------------- | ------------------------------------ |
| ITツール登録     | 補助対象となるITツールを事務局に登録 |
| 提案・導入       | 中小企業に適切なITツールを提案・導入 |
| 申請サポート     | 交付申請の共同作成・提出サポート     |
| アフターサポート | 導入後の運用支援・問い合わせ対応     |
| 実績報告サポート | 実績報告の確認・承認                 |
| 不正防止         | 補助事業の管理監督・不正行為の防止   |

---

## 登録形態

### 1. 法人（単独）

IT導入支援事業者の登録要件をすべて満たす法人が単独で登録する形態です。

- ITツール登録から効果報告まですべて1法人で実施
- 大手ITベンダー、SIer、ソフトウェアメーカー等が該当

### 2. コンソーシアム

幹事社1社と構成員からなる形態です。

```
【コンソーシアムの構成】

  ┌─────────────────────────────────────┐
  │            幹事社                   │
  │  ・事務局への登録・申請             │
  │  ・コンソーシアム全体の管理         │
  │  ・責任の所在                       │
  └─────────────────────────────────────┘
           │
    ┌──────┼──────┐
    ↓      ↓      ↓
  ┌────┐┌────┐┌────┐
  │構成員││構成員││構成員│
  │    ││    ││    │
  │販売 ││導入 ││保守 │
  │担当 ││担当 ││担当 │
  └────┘└────┘└────┘
```

#### コンソーシアムのメリット

- 単独で要件を満たせない法人・個人事業主も構成員として参画可能
- 役割分担による効率的な事業運営
- 地域密着型のサポート体制構築

## 登録要件

### 基本要件

- [ ] 日本国内で法人登記されている（または個人事業主）
- [ ] 安定的な事業基盤を有している
- [ ] 中小企業等に対するITツールの導入支援実績がある
- [ ] 情報セキュリティに関する対策を講じている
- [ ] 補助事業を適切に遂行できる体制を有している

### 禁止事項

- [ ] 補助事業者と同一法人・同一代表者ではないこと
- [ ] 反社会的勢力との関係がないこと
- [ ] 過去にIT導入補助金で登録取消を受けていないこと
- [ ] 不正行為を行っていないこと

## 登録手続き

### 登録スケジュール（2026年度予定）

| 項目         | 日程                             |
| ------------ | -------------------------------- |
| 登録申請開始 | 2026年3月30日(月) 10:00〜        |
| 登録申請締切 | 公募期間中（詳細は公募要領参照） |

### 登録手順

1. **仮登録**
   - 電子申請画面より仮登録
   - 事業者情報の入力

2. **本登録**
   - 仮登録完了メールのURLから本登録
   - 必要書類のアップロード
   - 審査

3. **ITツール登録**
   - IT事業者登録完了後、ITツールの登録
   - ツールごとに審査

### 必要書類

| 書類                       | 内容                          |
| -------------------------- | ----------------------------- |
| 履歴事項全部証明書         | 発行から3ヶ月以内             |
| 事業概要資料               | 会社案内等                    |
| 財務諸表                   | 直近1期分                     |
| 個人情報保護方針           | Webサイト等で公開していること |
| セキュリティ対策の実施状況 | 体制図等                      |

## ITツール登録

### 登録可能なITツールのカテゴリ

| 大分類               | 内容                                       |
| -------------------- | ------------------------------------------ |
| 大分類Ⅰ ソフトウェア | 業務パッケージ、クラウドサービス等         |
| 大分類Ⅱ オプション   | 機能拡張、カスタマイズ等                   |
| 大分類Ⅲ 役務         | 導入コンサルティング、研修、保守等         |
| 大分類Ⅳ ハードウェア | PC、タブレット、レジ等（インボイス枠のみ） |

### ITツール登録の審査ポイント

- [ ] 生産性向上に資するツールであること
- [ ] 継続的なサポート体制があること
- [ ] 適正な価格設定であること
- [ ] セキュリティ対策が講じられていること

## コンサルとしてのIT導入支援事業者活用ポイント

### IT導入支援事業者の選び方（顧客へのアドバイス）

```markdown
## IT導入支援事業者選定チェックリスト

### 基本確認

- [ ] IT導入補助金のIT導入支援事業者として登録されているか
- [ ] 導入したいITツールを取り扱っているか
- [ ] 自社の業種・規模への導入実績があるか

### サポート体制

- [ ] 導入前の相談・提案は無料か
- [ ] 導入時のサポート内容は明確か
- [ ] 導入後のサポート体制（保守・運用）は十分か
- [ ] 問い合わせ対応時間・方法は自社に合っているか

### 費用

- [ ] 見積もりの内訳は明確か
- [ ] 補助対象経費と対象外経費の説明があるか
- [ ] 追加費用の可能性は説明されているか

### 実績・信頼性

- [ ] 過去のIT導入補助金での採択実績
- [ ] 顧客事例・導入事例の紹介
- [ ] 会社の安定性（設立年数、従業員数等）
```

### 複数事業者への相談を推奨

- 最低3社から見積もり・提案を取得
- 価格だけでなくサポート内容を比較
- 担当者の対応・コミュニケーション力も重視

## 登録取消について

IT導入支援事業者として不適切な行為が確認された場合、登録取消となります。

### 登録取消の理由例

- 虚偽の申請・報告
- 不正行為への関与
- 補助事業者との共謀
- 事務局への虚偽報告
- 立入調査への非協力

### 登録取消事業者一覧

[IT導入支援事業者の登録取消について](https://it-shien.smrj.go.jp/pdf/deregistration_list.pdf)

> **注意**: 登録取消となった事業者と取引のある補助事業者は、交付決定取消・補助金返還の対象となる可能性があります。

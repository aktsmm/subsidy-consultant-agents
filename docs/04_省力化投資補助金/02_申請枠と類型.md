# 省力化投資補助金 申請枠と類型

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [省力化投資補助金公式サイト](https://shoryokuka.smrj.go.jp/)

---

## 2つの類型

### 1. カタログ注文型

簡易で即効性がある省力化投資を支援します。事前にカタログに登録された製品から選ぶため、申請が簡単です。

| 項目       | 内容                                         |
| ---------- | -------------------------------------------- |
| 対象       | カタログに掲載された省力化効果のある汎用製品 |
| 補助率     | 1/2（賃上げ要件達成で2/3）                   |
| 補助上限額 | 従業員規模により200万円〜1,500万円           |
| 申請方法   | **随時受付中**                               |

#### 補助上限額の詳細

| 従業員数 | 補助上限額（通常） | 補助上限額（賃上げ達成） |
| -------- | ------------------ | ------------------------ |
| 5人以下  | 200万円            | 300万円                  |
| 6〜20人  | 500万円            | 750万円                  |
| 21〜50人 | 1,000万円          | 1,500万円                |
| 51人以上 | 1,000万円          | 1,500万円                |

#### カタログ製品の例

| カテゴリ             | 製品例                     | 想定業種     |
| -------------------- | -------------------------- | ------------ |
| 配膳ロボット         | 飲食店向け自動配膳ロボット | 飲食業       |
| 清掃ロボット         | 業務用自動床洗浄機         | サービス業   |
| 検品・仕分けシステム | AI外観検査装置             | 製造業       |
| 無人決済システム     | セルフレジ、自動精算機     | 小売業       |
| 在庫管理システム     | RFID在庫管理               | 小売・卸売業 |
| 自動搬送ロボット     | AGV・AMR                   | 製造・物流業 |

---

### 2. 一般型

事業内容に合わせた多様な設備やシステム導入を支援します。

| 項目       | 内容                                                     |
| ---------- | -------------------------------------------------------- |
| 対象       | 個別現場の設備や事業内容に合わせた設備導入・システム構築 |
| 補助率     | 1/3〜2/3（従業員規模・賃上げ達成により変動）             |
| 補助上限額 | 最大1億円                                                |
| 申請方法   | 公募締切あり                                             |

#### 補助上限額の詳細

| 従業員数 | 補助上限額 |
| -------- | ---------- |
| 5人以下  | 1,000万円  |
| 6〜50人  | 5,000万円  |
| 51人以上 | 1億円      |

---

## カタログ注文型 vs 一般型の比較

| 項目           | カタログ注文型     | 一般型                 |
| -------------- | ------------------ | ---------------------- |
| 投資内容       | 簡易で即効性がある | オーダーメイド性がある |
| 補助対象       | カタログ掲載製品   | 個別設備・システム     |
| 補助上限       | 最大1,500万円      | 最大1億円              |
| 申請タイミング | 随時               | 公募期間内             |
| 審査難易度     | 比較的容易         | 事業計画審査あり       |
| 申請〜交付決定 | 約1ヶ月            | 約2〜3ヶ月             |

---

## 対象経費

| 経費区分       | カタログ型 | 一般型 | 内容                     |
| -------------- | :--------: | :----: | ------------------------ |
| 設備費         |     ✅     |   ✅   | 省力化製品・設備の購入費 |
| 導入経費       |     ✅     |   ✅   | 設置費、設定費           |
| システム構築費 |     -      |   ✅   | カスタマイズ、連携開発   |
| 外注費         |     -      |   ✅   | 導入コンサルティング     |

---

## 類型の選び方フローチャート

```
[導入したい製品は？]
    │
    ├── カタログに掲載されている既製品
    │   └── カタログ注文型 ✅
    │       ・随時申請可
    │       ・審査が容易
    │       ・最大1,500万円
    │
    └── 自社の現場に合わせたカスタム設備・システム
        └── 一般型 ✅
            ・公募期間内に申請
            ・事業計画書が必要
            ・最大1億円
```

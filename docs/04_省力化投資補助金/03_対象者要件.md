# 省力化投資補助金 対象者要件

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [省力化投資補助金公式サイト](https://shoryokuka.smrj.go.jp/)

---

## 対象となる事業者

### 中小企業の定義

| 業種                     | 資本金        | 従業員数  |
| ------------------------ | ------------- | --------- |
| 製造業、建設業、運輸業等 | 3億円以下     | 300人以下 |
| 卸売業                   | 1億円以下     | 100人以下 |
| サービス業               | 5,000万円以下 | 100人以下 |
| 小売業                   | 5,000万円以下 | 50人以下  |

---

## 基本要件

### 共通要件

- [ ] 中小企業・小規模事業者であること
- [ ] 人手不足の状態であること（または今後見込まれること）
- [ ] 省力化による生産性向上の計画があること

### 賃上げ要件（補助率・上限額の引上げ条件）

| 要件             | 内容                          |
| ---------------- | ----------------------------- |
| 事業場内最低賃金 | 地域別最低賃金+30円以上を達成 |
| 給与支給総額     | 年率平均1.5%以上増加          |

> 賃上げ要件を満たすと、カタログ型の補助率が1/2→2/3に、補助上限額も引き上がります。

---

## カタログ注文型の追加要件

| 要件               | 内容                                               |
| ------------------ | -------------------------------------------------- |
| カタログ製品の選定 | 事務局が登録したカタログ製品から選ぶこと           |
| 販売代理店         | カタログに記載された販売代理店を通じて購入すること |
| 省力化効果の見込み | 導入により人手不足解消の効果が見込まれること       |

---

## 一般型の追加要件

| 要件           | 内容                                             |
| -------------- | ------------------------------------------------ |
| 事業計画書     | 省力化の効果を定量的に示す事業計画を策定すること |
| 付加価値額     | 年率平均3%以上の付加価値額増加見込み             |
| gBizIDプライム | 電子申請に必要（jGrants使用）                    |

---

## 対象外となる事業者

- [ ] 大企業の子会社（みなし大企業）
- [ ] 直近3年間で省力化投資補助金の交付決定を受け、事業完了していない者
- [ ] 税金の未納がある者
- [ ] 反社会的勢力に関与している者

---

## 業種別の適合チェック

### 飲食業

| チェック項目 | 確認内容                             |
| ------------ | ------------------------------------ |
| 人手不足     | ホールスタッフ・調理補助の不足       |
| 適した製品   | 配膳ロボット、食器洗浄機、セルフレジ |
| 推奨類型     | カタログ注文型                       |

### 製造業

| チェック項目 | 確認内容                                      |
| ------------ | --------------------------------------------- |
| 人手不足     | 検品・組立・梱包等の人手不足                  |
| 適した製品   | AI検査装置、産業用ロボット、AGV               |
| 推奨類型     | カタログ型（既製品）or 一般型（カスタム設備） |

### 小売業

| チェック項目 | 確認内容                               |
| ------------ | -------------------------------------- |
| 人手不足     | レジ・在庫管理・清掃等の人手不足       |
| 適した製品   | セルフレジ、RFID在庫管理、清掃ロボット |
| 推奨類型     | カタログ注文型                         |

### 物流業

| チェック項目 | 確認内容                           |
| ------------ | ---------------------------------- |
| 人手不足     | 仕分け・ピッキング・搬送の人手不足 |
| 適した製品   | 自動仕分けシステム、AGV、自動倉庫  |
| 推奨類型     | 一般型（大規模な場合）             |

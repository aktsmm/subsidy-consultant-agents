# 省力化投資補助金 概要

> **情報更新日**: 2026年1月28日  
> **正式名称**: 中小企業省力化投資補助金  
> **公式サイト**: [省力化投資補助金](https://shoryokuka.smrj.go.jp/)

---

## 制度の仕組み

### 補助金の目的

人手不足に悩む中小企業等に対して、**IoT・ロボット等の人手不足解消に効果がある汎用製品**の導入、または**個別の現場や事業内容等に合わせた設備導入・システム構築**等の省力化投資を支援する補助金です。

中小企業等の**売上拡大や生産性向上**を後押しし、**付加価値額や生産性向上**を図り、**賃上げ**につなげることを目的としています。

### 2つの類型

| 類型           | 特徴                           | 補助上限      | 申請方法       |
| -------------- | ------------------------------ | ------------- | -------------- |
| カタログ注文型 | 簡易で即効性がある省力化製品   | 最大1,500万円 | **随時受付中** |
| 一般型         | オーダーメイドの設備・システム | 最大1億円     | 公募期間内     |

## 他補助金との比較

| 項目       | 省力化投資補助金     | IT導入補助金          | ものづくり補助金 |
| ---------- | -------------------- | --------------------- | ---------------- |
| 対象       | ハードウェア中心     | ソフトウェア中心      | 設備投資全般     |
| 目的       | 人手不足解消・省力化 | 業務効率化・DX        | 革新的製品開発   |
| 補助上限額 | 最大1億円            | 最大450万円（通常枠） | 最大4,000万円    |
| カタログ型 | あり（随時申請可）   | なし                  | なし             |
| 審査難易度 | カタログ型は容易     | 比較的容易            | 厳格             |

### 使い分けの目安

| ニーズ                           | 推奨補助金                           |
| -------------------------------- | ------------------------------------ |
| ロボット・自動化機器を導入したい | **省力化投資補助金**                 |
| 業務ソフトウェアを導入したい     | IT導入補助金                         |
| 製造設備を導入したい             | 省力化投資補助金 or ものづくり補助金 |
| インボイス対応したい             | IT導入補助金                         |
| 配膳ロボットを導入したい         | **省力化投資補助金（カタログ型）**   |

## コンサルとしての主要チェックポイント

### 初回ヒアリング時

- [ ] 人手不足の状況の確認
- [ ] 導入したい省力化製品・設備の明確化
- [ ] カタログ型 or 一般型どちらが適切か判断
- [ ] 従業員数の確認（補助上限額に影響）
- [ ] 賃上げ計画の有無（補助率・上限に影響）

### カタログ型の場合

- [ ] カタログ掲載製品から選定
- [ ] 販売代理店の確認
- [ ] 随時申請のため、早期着手を推奨

### 一般型の場合

- [ ] 事業計画書の策定
- [ ] 公募期間の確認
- [ ] 設備の仕様・見積りの準備

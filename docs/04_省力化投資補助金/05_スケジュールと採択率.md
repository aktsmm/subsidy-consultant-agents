## スケジュール

### カタログ注文型

| 項目     | 内容              |
| -------- | ----------------- |
| 申請受付 | **随時受付中**    |
| 交付決定 | 申請から約1ヶ月   |
| 事業期間 | 交付決定〜約6ヶ月 |

> **メリット**: 締切を気にせず、いつでも申請可能です。

## 採択率

### カタログ注文型

| 指標   | 値                                                         |
| ------ | ---------------------------------------------------------- |
| 採択率 | **約70〜80%**（推定）                                      |
| 傾向   | カタログ製品を選ぶだけなので、書類不備がなければ高い採択率 |

### 一般型

| 指標   | 値                                           |
| ------ | -------------------------------------------- |
| 採択率 | **約45〜55%**（推定）                        |
| 傾向   | ものづくり補助金と同程度、事業計画の質が重要 |

## 注意点

### 1. カタログ型は掲載製品のみ

カタログに掲載されていない製品は対象外です。導入したい製品がカタログにない場合は、一般型で申請するか、他の補助金（ものづくり補助金等）を検討してください。

### 2. 省力化の効果を示す必要あり

「便利だから導入したい」では不十分です。**何人分の工数を削減できるか**、**生産性がどの程度向上するか**を具体的に示す必要があります。

### 3. 賃上げ要件の達成義務

賃上げ要件で補助率・上限額の引上げを受けた場合、事業完了後に達成状況が確認されます。未達成の場合、補助金の一部返還を求められる可能性があります。

### 4. 交付決定前のリース契約も不可

リース契約も交付決定前に締結すると対象外になります。リースでの導入を検討している場合は、リース会社との事前調整が必要です。

## FAQ

### Q: カタログにない製品は追加できますか？

**A**: 製品メーカーが事務局にカタログ登録申請を行うことで、追加される場合があります。導入したい製品がカタログにない場合は、メーカーに登録を依頼するか、一般型で申請してください。

### Q: IT導入補助金との併用は？

**A**: 同一経費での併用は不可ですが、ハードウェア（省力化投資補助金）とソフトウェア（IT導入補助金）で分けて申請することは可能です。

### Q: カタログ型と一般型の同時申請は？

**A**: 異なる設備であれば可能です。ただし、同一設備での併用は不可です。

### Q: リースでの導入は対象ですか？

**A**: カタログ型はリースも対象となる場合があります。一般型は原則購入が対象ですが、条件によりリースも認められる場合があります。公募要領で確認してください。

# 省力化投資補助金 申請フロー

> **情報更新日**: 2026年1月28日  
> **情報ソース**: [省力化投資補助金公式サイト](https://shoryokuka.smrj.go.jp/)

---

## カタログ注文型の申請フロー

```
STEP1: カタログ製品の検索・選定
  ├── 公式サイトの製品カタログを検索
  ├── 業種・用途・予算で絞り込み
  └── 販売代理店への問い合わせ
    ↓
STEP2: 販売代理店との商談
  ├── 製品の仕様・価格の確認
  ├── 導入スケジュールの確認
  └── 見積書の取得
    ↓
STEP3: 交付申請（電子申請）
  ├── 申請フォームに必要事項を入力
  ├── 見積書・導入計画書等を添付
  └── 申請提出
    ↓
STEP4: 審査・交付決定（約1ヶ月）
    ↓
STEP5: 製品の契約・導入 ※交付決定後に契約
  ├── 販売代理店と契約
  ├── 製品の納品・設置
  └── 支払い
    ↓
STEP6: 実績報告
  ├── 導入の証拠書類提出
  └── 省力化効果の報告
    ↓
STEP7: 補助金交付
```

> **ポイント**: カタログ型は**随時受付**のため、いつでも申請可能です。審査も約1ヶ月と短いのが特徴です。

---

## 一般型の申請フロー

```
STEP1: 事前準備（1〜2ヶ月前）
  ├── gBizIDプライム取得
  ├── 導入設備の検討・見積り取得
  └── 事業計画のドラフト作成
    ↓
STEP2: 事業計画書の策定（1ヶ月前）
  ├── 省力化効果の定量化
  ├── 付加価値額の増加見込み計算
  └── 実施体制・スケジュールの策定
    ↓
STEP3: 交付申請（jGrants）
  ├── 事業計画書のアップロード
  ├── 見積書・各種添付書類
  └── 申請提出
    ↓
STEP4: 審査・交付決定（約2〜3ヶ月）
    ↓
STEP5: 補助事業の実施 ※交付決定後に契約
  ├── 設備の契約・発注
  ├── システム構築・設置
  └── 支払い完了
    ↓
STEP6: 実績報告
  ├── 経費の証拠書類提出
  ├── 省力化効果の実績報告
  └── 設備導入の before/after 写真
    ↓
STEP7: 確定検査・補助金交付
    ↓
STEP8: 効果報告（年次・3〜5年間）
```

## 申請に必要な書類

### カタログ注文型

| 書類         | 内容                   |
| ------------ | ---------------------- |
| 申請書       | 基本情報・導入計画     |
| 見積書       | 販売代理店からの見積り |
| 導入計画書   | 省力化効果の見込み     |
| 確定申告書等 | 事業者の確認書類       |

### 一般型

| 書類                | 内容                         |
| ------------------- | ---------------------------- |
| 事業計画書          | 省力化の計画・数値目標       |
| 見積書（2社以上）   | 50万円以上の設備は比較見積り |
| 決算書（直近2期分） | 財務状況の確認               |
| 付加価値額計算書    | 基本要件の達成見込み         |
| 導入設備の仕様書    | 設備の詳細スペック           |

## 💡 最重要注意事項

### 交付決定前の契約禁止

> **⚠️ カタログ型・一般型ともに、交付決定前の契約・発注・支払いは補助対象外です。**

```
【OK】交付決定通知 → 契約 → 納品 → 支払い
【NG】契約 → 交付決定通知 → 納品 → 支払い（全額自己負担に！）
```

### 証拠書類の保管

すべての取引について以下の書類を揃える必要があります：

- 見積書
- 契約書（注文書・注文請書）
- 納品書
- 請求書
- 振込明細（支払い証拠）
- 設備の写真（設置前後）

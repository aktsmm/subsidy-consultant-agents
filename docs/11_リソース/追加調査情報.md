## 1. 中小企業の定義・みなし大企業判定基準

### 中小企業基本法の定義

| 業種分類                         | 資本金        | 従業員    |
| -------------------------------- | ------------- | --------- |
| 製造業その他                     | 3億円以下     | 300人以下 |
| 卸売業                           | 1億円以下     | 100人以下 |
| 小売業                           | 5,000万円以下 | 50人以下  |
| サービス業                       | 5,000万円以下 | 100人以下 |
| ゴム製品製造業（一部除く）       | 3億円以下     | 900人以下 |
| ソフトウェア・情報処理サービス業 | 3億円以下     | 300人以下 |
| 旅館業                           | 5,000万円以下 | 200人以下 |

### 小規模事業者の定義

| 業種分類                           | 従業員   |
| ---------------------------------- | -------- |
| 商業・サービス業（宿泊・娯楽除く） | 5人以下  |
| サービス業のうち宿泊・娯楽業       | 20人以下 |
| 製造業その他                       | 20人以下 |

### みなし大企業（対象外となるケース）

多くの補助金で「みなし大企業」は対象外。詳細は各制度の公募要領で確認。

**参照**: https://www.chusho.meti.go.jp/soshiki/teigi.html

## 2. 省力化投資補助金 詳細

### 2つの類型

| 類型               | 特徴                       | 補助上限      |
| ------------------ | -------------------------- | ------------- |
| **カタログ注文型** | カタログから汎用製品を選択 | 最大1,500万円 |
| **一般型**         | オーダーメイドの設備投資   | 最大1億円     |

### カタログ注文型 補助上限額（従業員数別）

| 従業員数 | 補助上限額 | 賃上げ達成時 |
| -------- | ---------- | ------------ |
| 5名以下  | 200万円    | 300万円      |
| 6〜20名  | 500万円    | 750万円      |
| 21名以上 | 1,000万円  | 1,500万円    |

補助率: 1/2以下

## 3. ものづくり補助金 詳細（22次締切）

### 申請枠

| 枠                           | 補助上限      | 補助率   |
| ---------------------------- | ------------- | -------- |
| 製品・サービス高付加価値化枠 | 最大4,000万円 | 1/2〜2/3 |
| グローバル枠                 | 最大4,000万円 | 1/2〜2/3 |

### スケジュール（22次締切）

| 日程                 | 内容         |
| -------------------- | ------------ |
| 2025年10月24日       | 公募要領公開 |
| 2025年12月26日 17:00 | 申請開始     |
| 2026年1月30日 17:00  | 申請締切     |

### 基本要件

1. 付加価値額：年率+3%以上
2. 給与支給総額：年率+1.5%以上
3. 事業所内最低賃金：地域別最低賃金+30円以上
4. 従業員の仕事・子育て両立要件（新設）

### 主な加点項目

- 賃上げ加点（大幅な賃上げ計画）
- 最低賃金引上げ加点
- 経営革新計画承認
- パートナーシップ構築宣言
- 災害対策要件

**参照**: https://portal.monodukuri-hojo.jp/about.html

## 9. 補助金併用ルール

### 一般原則

- 同一事業・同一経費に対して複数の補助金を重複して受けることは不可
- 異なる経費・異なる事業であれば併用可能な場合あり
- 各補助金の公募要領で個別に確認が必要

### 注意点

- 国庫補助金同士の重複禁止
- 過去に補助金を受けた場合の減点・制限
- 補助金受給後の財産処分制限
